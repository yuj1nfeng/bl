## 预算规划的重要性

> "没钱做不出好作品，但乱花钱一定做不出好作品。"

科学的预算规划能帮助你：

- **控制成本** - 避免超支风险
- **资源优化** - 把钱花在刀刃上
- **风险预判** - 提前预留应急资金
- **融资依据** - 给投资方清晰预期

## 一、漫剧成本构成

### 成本分类总览

```
                    总成本
                       │
        ┌──────────────┼──────────────┐
        ↓              ↓              ↓
    前期成本        制作成本        后期成本
   (10-15%)       (60-70%)        (15-20%)
        │              │              │
   策划/剧本      美术/动画       剪辑/特效
```

## 二、预算编制方法

### 1. 自上而下法

先确定总预算，再分配到各项目。

**适用场景：**
- 投资金额已确定
- 预算有上限

**分配比例参考：**
```
总预算：100%
├── 前期制作：12%
├── 动画制作：65%
├── 后期制作：15%
└── 预备金：8%
```

### 2. 自下而上法

先估算各项目成本，再汇总成总预算。

**适用场景：**
- 项目需求明确
- 需要精确核算

**计算步骤：**
```
Step 1: 列出所有工作任务
Step 2: 估算每项工作所需工时
Step 3: 确定各岗位日薪/时薪
Step 4: 计算人力成本
Step 5: 加上设备/场地/其他成本
Step 6: 加上预备金（10-15%）
```

### 3. 类比估算法

参考类似项目的实际成本进行估算。

**适用场景：**
- 缺乏详细数据
- 需要快速估算

**注意事项：**
- 选择真正可比的项目
- 考虑通胀和行情变化
- 根据自身情况调整

## 三、预算控制技巧

### 1. 建立预算跟踪表

```
预算跟踪表模板:
┌──────────┬────────┬────────┬────────┬────────┐
│ 项目     │ 预算   │ 实际   │ 差异   │ 状态   │
├──────────┼────────┼────────┼────────┼────────┤
│ 剧本     │ 15,000 │ 14,500 │ +500   │ ✓      │
│ 人设     │ 20,000 │ 22,000 │ -2,000 │ ⚠      │
│ 动画     │ 50,000 │ 48,000 │ +2,000 │ ✓      │
│ ...      │ ...    │ ...    │ ...    │ ...    │
└──────────┴────────┴────────┴────────┴────────┘
```

### 2. 设置预警机制

```
预警级别:
├── 绿色（正常）- 实际/预算 < 90%
├── 黄色（关注）- 90% ≤ 实际/预算 < 100%
├── 橙色（警告）- 100% ≤ 实际/预算 < 110%
└── 红色（超支）- 实际/预算 ≥ 110%
```

### 3. 变更管理流程

```
预算变更流程:
申请 → 评估 → 审批 → 执行 → 记录

变更原因分类:
□ 需求变更
□ 技术难题
□ 人员变动
□ 外部因素
□ 其他
```

### 4. 预留应急资金

```
预备金设置:
├── 基础预备金：总预算的 5-10%
├── 风险预备金：总预算的 5%
└── 使用条件：需项目负责人审批
```

## 四、团队组建

### 1. 核心岗位设置

**最小可行团队（3-5 人）**
```
┌─────────────────────────────────────┐
│           导演/制片                 │
│    (统筹全局、创意决策)             │
└─────────────────────────────────────┘
                  │
    ┌─────────────┼─────────────┐
    ↓             ↓             ↓
┌───────┐   ┌─────────┐   ┌─────────┐
│ 编剧  │   │ 美术/动画│   │ 后期/音效│
│ 1 人  │   │ 1-2 人  │   │ 1 人    │
└───────┘   └─────────┘   └─────────┘
```

**标准团队（8-12 人）**
```
                    制片人
                      │
    ┌─────────────────┼─────────────────┐
    ↓                 ↓                 ↓
  导演              制片              监制
    │                 │                 │
    ├─→ 编剧 (1-2 人)  │                 │
    │                 │                 │
    ├─→ 美术组        │                 │
    │   ├─ 角色设计   │                 │
    │   └─ 场景设计   │                 │
    │                 │                 │
    ├─→ 动画组        │                 │
    │   ├─ 原画师     │                 │
    │   ├─ 动画师     │                 │
    │   └─ 上色师     │                 │
    │                 │                 │
    ├─→ 声音组        │                 │
    │   ├─ 配音导演   │                 │
    │   ├─ 音效师     │                 │
    │   └─ 配乐师     │                 │
    │                 │                 │
    └─→ 后期组                          │
        ├─ 剪辑师                       │
        └─ 特效师                       │
```

### 2. 岗位职责详解

**导演**
```
职责:
□ 创意决策和整体把控
□ 指导各环节创作

能力要求:
□ 审美能力
□ 沟通能力

□ 行业经验 3 年+
```

**编剧**
```
职责:
□ 剧本创作和修改

□ 对白设计

能力要求:
□ 故事架构能力
□ 文字功底
□ 理解视觉语言

```

```
能力要求:
□ 运动规律理解
```

**后期师**

```
能力要求:
□ 精通 AE/PR
```

### 4. 合作模式

**全职雇佣**
```
优点:
✓ 稳定性高
✓ 沟通成本低
✓ 忠诚度高

缺点:
✗ 成本高（社保 + 福利）
✗ 灵活性低

适用: 核心岗位、长期项目
```

**兼职合作**
```
优点:
✓ 成本可控
✓ 灵活调配
✓ 专业对口

缺点:
✗ 时间协调难
✗ 稳定性低

适用: 非核心岗位、阶段性工作
```

**外包合作**
```
优点:
✓ 专业团队
✓ 交付明确
✓ 降低管理成本

缺点:
✗ 沟通成本高
✗ 质量把控难
✗ 利润被分走

适用: 标准化工作、产能不足时
```

### 2. 工作流程规范

**文件命名规范：**
```
命名规则:
项目_类型_内容_版本_日期.格式

示例:
MH_Script_EP01_v02_20260306.docx
MH_Char_Main_A_v03_20260306.psd
MH_Scene_Cafe_Day_v01_20260306.psd
```

**版本管理规范：**
```
版本命名:
v0.x - 草稿/内部版本
v1.0 - 正式版本
v1.1 - 小修改
v2.0 - 大修改

存档规则:
- 保留最近 3 个版本
- 正式版永久保存
- 定期清理临时文件
```

**审核流程：**
```
审核流程:
制作 → 自检 → 组长审核 → 导演审核 → 修改 → 终稿

审核要点:
□ 符合需求
□ 质量达标
□ 按时交付
□ 格式规范
```

## 总结

预算规划和团队组建是项目成功的保障。关键要点：

1. **预算合理** - 根据项目规模科学规划
2. **预留弹性** - 预备金应对突发情况
3. **团队适配** - 根据项目需求配置人员
4. **管理规范** - 建立标准化工作流程
5. **沟通顺畅** - 保持信息透明同步
