## 中间画基础

### 什么是中间画

**定义：**
- 原画之间的过渡帧
- 补充动作过程
- 保证流畅度

**作用：**
- 连接关键帧
- 控制动作节奏
- 保证质量稳定

### 中间画数量

**决定因素：**
- 动作速度
- 流畅度要求
- 制作成本

**常见数量：**
```
快速动作：1-2 张中间画
正常动作：3-5 张中间画
慢速动作：5-7 张中间画
```

## 中间画绘制技巧

### 对位技巧

**定位器使用：**
1. 放置定位器
2. 对齐原画
3. 绘制中间画
4. 检查对位

**检查要点：**
- 轮廓对齐
- 关键点对齐
- 无偏移抖动

### 中间点确定

**等分法：**
- 找到起点终点
- 等分距离
- 确定中间位置

**轨迹法：**
- 确定运动轨迹
- 沿轨迹等分
- 确定中间点

### 线条处理

**线条要求：**
- 流畅无断点
- 粗细一致
- 闭合完整

**线条衔接：**
- 注意线条走向
- 保持线条风格
- 避免线条抖动

## 常见动作中间画

### 走路中间画

**关键 pose：**
```
1. 接触位置
2. 下降位置
3. 经过位置
4. 上升位置
```

**中间画要点：**
- 重心起伏
- 手臂摆动
- 脚部滚动

### 跑步中间画

**关键 pose：**
```
1. 接触位置
2. 下降位置
3. 腾空位置
4. 经过位置
```

**中间画要点：**
- 腾空阶段
- 身体前倾
- 手臂摆动幅度大

### 转头中间画

**关键 pose：**
- 起始位置
- 中间位置
- 结束位置

**中间画要点：**
- 弧线运动
- 五官变形
- 头发跟随

## 中间画检查

### 动画检查

**检查方法：**
- 翻动检查
- 软件预览
- 多人审核

**检查要点：**
□ 动作流畅
□ 无抖动
□ 节奏正确
□ 符合原画

### 常见问题

**问题 1：抖动**

**原因：** 对位不准

**解决：**
- 重新对位
- 使用定位器
- 减少线条变化

**问题 2：漂浮感**

**原因：** 中间画太少

**解决：**
- 增加中间画
- 调整间距
- 添加缓入缓出

**问题 3：机械感**

**原因：** 匀速运动

**解决：**
- 调整中间画间距
- 添加缓入缓出
- 改为弧线运动
