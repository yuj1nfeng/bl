## 人体结构基础

### 人体比例

**标准比例：**
- 成年人：7-8 头身
- 青少年：6-7 头身
- 儿童：4-5 头身

**关键节点：**
- 头部
- 肩部
- 肘部
- 手腕
- 腰部
- 膝盖
- 脚踝

### 关节活动范围

**头部：** 左右旋转各 90 度
**肩部：** 360 度旋转
**肘部：** 弯曲 150 度
**腰部：** 左右各 30 度

## 动态绘制

### 动态线

**定义：** 表现动作趋势的辅助线
**作用：** 确定动作方向和力度

**绘制步骤：**
```
1. 确定动作意图
2. 画动态线
3. 添加体块
4. 细化轮廓
```

### 动作分解

**走路循环：**
```
4 个关键 pose:
1. 接触位置
2. 下降位置
3. 经过位置
4. 上升位置
```

**跑步循环：**
```
4 个关键 pose:
1. 接触位置
2. 下降位置
3. 腾空位置
4. 经过位置
```

### 重量感表现

**重心：**
- 站立：两脚之间
- 走路：前后脚之间
- 跑步：前脚

**平衡：**
- 静态平衡
- 动态平衡

## 常见动作

### 走路

**要点：**
- 重心上下起伏
- 手臂自然摆动
- 脚部滚动动作

**常见错误：**
- 滑行（脚不接触地面）
- 僵硬（没有起伏）
- 不协调（手脚同边）

### 跑步

**要点：**
- 身体前倾
- 腾空阶段
- 手臂摆动幅度大

**常见错误：**
- 速度不够
- 动作僵硬
- 重心不对

### 跳跃

**要点：**
- 预备动作（下蹲）
- 起跳动作
- 腾空动作
- 落地动作

### 坐姿

**要点：**
- 臀部受力
- 腿部弯曲
- 背部姿态
