## 特效绘制

### 光效绘制

**辉光效果：**
```
步骤:
1. 绘制基础形状
2. 添加高斯模糊
3. 调整混合模式（滤色/相加）
4. 多层叠加增强
```

**光束效果：**
```
步骤:
1. 画光束路径
2. 渐变填充
3. 添加粒子
4. 动态模糊
```

### 粒子特效

**粒子类型：**
- 光点粒子
- 烟雾粒子
- 火焰粒子
- 魔法粒子

**绘制方法：**
```
1. 确定发射源
2. 设置粒子属性
3. 绘制运动轨迹
4. 添加动态效果
```

### 爆炸特效

**爆炸结构：**
```
核心：最亮部分
中间：火焰层
外层：烟雾层
外围：冲击波
```

**绘制步骤：**
```
1. 画爆炸核心
2. 添加火焰
3. 绘制烟雾
4. 冲击波扩散
```

## 特效动画

### 动画原理

**生长动画：**
- 从小到大
- 从弱到强
- 时间 6-12 帧

**消散动画：**
- 从强到弱
- 逐渐透明
- 时间 12-24 帧

**循环动画：**
- 火焰循环
- 水流循环
- 魔法循环

### 时间控制

**快速特效：** 3-6 帧
**中速特效：** 6-12 帧
**慢速特效：** 12-24 帧
