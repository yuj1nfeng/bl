## 输出基础

### 视频参数

**分辨率：**
- 1080p：1920x1080
- 720p：1280x720
- 4K：3840x2160

**帧率：**
- 24fps：电影感
- 25fps：PAL 制式
- 30fps：NTSC 制式
- 60fps：流畅动作

**码率：**
- 1080p：6000-8000 kbps
- 720p：3000-5000 kbps
- 4K：20000-30000 kbps

### 音频参数

**格式：** AAC
**采样率：** 48kHz
**比特率：** 128-320 kbps
**声道：** 立体声

## 平台规范

### B 站

**视频要求：**
```
分辨率：1920x1080 或 3840x2160
帧率：24/25/30/60 fps
码率：6000-8000 kbps（1080p）
格式：MP4（H.264）
时长：无限制
大小：<4GB
```

**音频要求：**
```
格式：AAC
码率：320 kbps
采样率：48kHz
```

**封面要求：**
```
尺寸：1146x712px
比例：16:9
大小：<2MB
格式：JPG/PNG
```

### 抖音

**视频要求：**
```
分辨率：1080x1920（竖屏）
        1920x1080（横屏）
帧率：24/30/60 fps
码率：3000-6000 kbps
格式：MP4（H.264）
时长：1-5 分钟
大小：<500MB
```

**封面要求：**
```
尺寸：1080x1920px
比例：9:16
大小：<5MB
```

### 快手

**视频要求：**
```
分辨率：1080x1920 或 1920x1080
帧率：24/30 fps
码率：3000-5000 kbps
格式：MP4
时长：1-5 分钟
```

### 视频号

**视频要求：**
```
分辨率：1080x1920 或 1920x1080
帧率：24/30 fps
码率：3000-6000 kbps
格式：MP4
时长：1-10 分钟
大小：<1GB
```

### YouTube

**视频要求：**
```
分辨率：1920x1080 或 3840x2160
帧率：24/30/60 fps
码率：8000-12000 kbps（1080p）
格式：MP4（H.264/H.265）
时长：无限制
大小：<128GB
```

**音频要求：**
```
格式：AAC
码率：384 kbps
采样率：48kHz
```

## 输出设置

### Premiere 输出

**H.264 设置：**
```
格式：H.264
预设：匹配源 - 高比特率
Profile: High
Level: 5.1
关键帧：2 秒间隔
编码：VBR 2 次
```

### After Effects 输出

**渲染设置：**
```
格式：QuickTime
编解码器：H.264
质量：100%
分辨率：完整
帧率：匹配项目
```

### 剪映输出

**输出设置：**
```
分辨率：1080p
帧率：30fps
码率：推荐
格式：MP4
```

## 质量控制

### 输出前检查

□ 分辨率正确
□ 帧率正确
□ 时长正确
□ 音量达标
□ 无黑场
□ 无错误帧
□ 字幕正确
□ 文件命名规范

### 多设备测试

**测试设备：**
- 电脑
- 手机
- 平板
- 电视

**测试内容：**
- 画面质量
- 音频质量
- 字幕显示
- 播放流畅

### 问题排查

**画面问题：**
- 模糊：检查分辨率和码率
- 卡顿：检查帧率和码率
- 色偏：检查色彩空间

**音频问题：**
- 爆音：检查音量
- 不同步：检查采样率
- 杂音：检查降噪

## 文件管理

### 命名规范

```
项目_集数_版本_日期。格式

示例:
MH_EP01_Final_20260319.mp4
MH_EP01_Preview_20260319.mp4
```

### 版本管理

**版本命名：**
- v0.x：草稿版本
- v1.0：正式版本
- v1.1：小修改
- v2.0：大修改

### 备份策略

**3-2-1 原则：**
- 3 份副本
- 2 种介质
- 1 份异地
