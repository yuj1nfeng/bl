## 字幕基础

### 字幕类型

**对白字幕：** 角色对话内容
**说明字幕：** 场景说明、时间地点
**特效字幕：** 强调、夸张效果
**片头片尾：** 演职员表

### 字幕要素

**内容：** 准确传达信息
**字体：** 符合视频风格
**颜色：** 清晰可读
**位置：** 不遮挡画面

## 字体选择

### 字体类型

**宋体类：**
- 特点：传统、正式
- 适用：古风、正式场合

**黑体类：**
- 特点：现代、简洁
- 适用：现代、通用

**圆体类：**
- 特点：可爱、柔和
- 适用：轻松、可爱风格

**书法体：**
- 特点：艺术、个性
- 适用：标题、特殊场景

### 字体搭配

**原则：**
- 不超过 3 种字体
- 风格统一
- 层次分明

**示例：**
```
标题：书法体/粗黑体
正文：常规黑体
注释：细黑体/宋体
```

## 字幕设计

### 颜色搭配

**基本原则：**
- 与画面对比明显
- 白色最常用
- 黑色描边增强对比

**特殊情况：**
```
亮背景：黑字白边
暗背景：白字黑边
彩色背景：白字黑边
```

### 大小设置

**标准大小：**
- 1080p：40-50px
- 720p：30-40px
- 4K：60-80px

**层级大小：**
```
主标题：60px
副标题：40px
正文：30px
注释：24px
```

### 位置安排

**安全区域：**
- 底部 15%-20%
- 左右留边 10%
- 不遮挡关键画面

**多行字幕：**
```
最多 2 行
行间距 1.5 倍
居中对齐
```

## 字幕动画

### 入场动画

**淡入：**
- 透明度 0→100%
- 时长 0.3-0.5 秒
- 最常用

**滑入：**
- 从下向上
- 时长 0.3-0.5 秒
- 动感强

**缩放：**
- 从小到大
- 时长 0.2-0.4 秒
- 强调效果

### 出场动画

**淡出：**
- 透明度 100%→0
- 时长 0.3-0.5 秒

**滑出：**
- 向下/向上
- 时长 0.3-0.5 秒

### 强调动画

**跳动：**
- 上下跳动
- 强调重点

**放大：**
- 瞬间放大
- 吸引注意

**变色：**
- 颜色变化
- 强调内容

## 字幕制作

### 制作流程

```
1. 整理台词
2. 打轴（时间轴）
3. 设计样式
4. 添加动画
5. 检查校对
```

### 打轴技巧

**时间控制：**
- 每句 2-5 秒
- 留出阅读时间
- 配合语音节奏

**断句原则：**
- 按语义断句
- 避免破句
- 标点符号
