## 好对白的标准

### 基本要求

**符合人物：**
- 符合身份地位
- 符合性格特点
- 符合教育背景
- 符合时代背景

**推动剧情：**
- 传递信息
- 揭示关系
- 制造冲突
- 推动发展

**自然流畅：**
- 口语化表达
- 有停顿打断
- 有潜台词
- 有情感色彩

## 对话写作技巧

### 个性化对白

**词汇选择：**
```
知识分子：用词精准，句式完整
市井小民：口语化，俚语多
年轻人：网络用语，流行语
老年人：传统表达，经验丰富
```

**句式特点：**
```
急性子：短句，直接
慢性子：长句，委婉
自信者：肯定句多
自卑者：疑问句多
```

### 潜台词运用

**什么是潜台词：**
- 表面说的 vs 实际想的
- 言外之意
- 话里有话

**示例：**
```
表面："你最近很忙啊"
潜台词："你都不陪我"

表面："随便你"
潜台词："我很在意"

表面："他挺好的"
潜台词："我不喜欢他"
```

### 对话节奏

**快慢变化：**
```
争吵：快，打断，重叠
谈心：慢，停顿，思考
紧张：快，简短
放松：慢，随意
```

**停顿运用：**
```
思考停顿："我...不知道"
情感停顿："我...爱你"
戏剧停顿："你是...凶手？"
```

## 常见问题与解决

### 问题 1：对白太书面

**症状：** 像念稿，不自然

**解决：**
- 大声朗读
- 录音听效果
- 加入口语词
- 允许不完整句

### 问题 2：信息倾倒

**症状：** 大段解释性对白

**解决：**
- 拆分信息
- 用动作代替
- 让观众推测
- 信任观众智商

### 问题 3：千人一面

**症状：** 所有角色说话一样

**解决：**
- 为每个角色设计口头禅
- 区分教育背景
- 区分性格特点
- 区分年龄层次

## 对话练习

### 练习 1：改写练习

**任务：** 把书面语改成口语

**示例：**
```
书面：我非常感谢你今天的帮助
口语：今天真多亏你了
```

### 练习 2：潜台词练习

**任务：** 写一段有潜台词的对话

**情境：** 情侣吵架，但都不说真实想法

### 练习 3：个性化练习

**任务：** 同一句话，5 个不同角色说

**例句：** "我不想去"

**角色：**
- 任性大小姐
- 老实上班族
- 叛逆少年
- 温柔妈妈
- 霸道总裁
