## 音效同步基础

### 同步重要性

**真实感：** 音画同步增强真实感
**冲击力：** 精确同步增强冲击
**专业度：** 同步准确体现专业

### 同步精度

**电影标准：** ±1 帧（40ms）
**电视标准：** ±2 帧（80ms）
**网络视频：** ±3 帧（120ms）

## 音效类型

### 动作音效

**同步要点：**
- 精确到帧
- 提前准备
- 延音处理

**常见音效：**
- 脚步声
- 衣物摩擦
- 物品碰撞
- 身体动作

### 环境音效

**同步要点：**
- 持续铺垫
- 音量适中
- 场景匹配

**常见音效：**
- 室内环境
- 室外环境
- 特殊环境

### 特殊音效

**同步要点：**
- 精确同步
- 层次丰富
- 动态处理

**常见音效：**
- 魔法效果
- 科幻音效
- 爆炸效果

## 同步技巧

### 波形对齐

**步骤：**
```
1. 导入音频和视频
2. 显示音频波形
3. 找到动作峰值
4. 对齐音效峰值
```

**技巧：**
- 放大时间轴
- 找到清晰峰值
- 微调位置

### 提前量处理

**原理：** 声音传播需要时间

**提前量：**
- 近距离：0 帧
- 中距离：1 帧
- 远距离：2-3 帧

**应用：**
- 雷声（先闪电后雷声）
- 远枪声（先见光后听声）

### 延音处理

**原理：** 声音有衰减过程

**处理：**
- 动作结束声音继续
- 自然衰减
- 混响效果

**示例：**
```
关门声:
- 撞击声（0 帧）
- 震动声（+2 帧）
- 余音（+10 帧）
```

## 分层同步

### 多层音效

**示例：爆炸音效**
```
层 1：爆炸核心（0 帧）
层 2：冲击波（+1 帧）
层 3：碎片（+2 帧）
层 4：余音（+5 帧）
```

### 音量平衡

**原则：**
- 对白优先
- 音效次之
- BGM 最后

**音量参考：**
- 对白：-6dB
- 主音效：-12dB
- 次音效：-18dB
- BGM：-24dB

## 软件操作

### Premiere 同步

**自动同步：**
```
1. 选择视频和音频
2. 右键 → 同步
3. 选择同步点（音频波形）
4. 自动对齐
```

**手动同步：**
```
1. 显示音频波形
2. 拖动对齐
3. 微调位置
```

### After Effects 同步

**标记点对齐：**
```
1. 在动作帧添加标记
2. 在音效峰值添加标记
3. 对齐标记
```

**表达式同步：**
```
将音效绑定到动作
使用表达式控制
```

### 剪映同步

**自动对齐：**
- 导入素材
- 自动识别
- 一键同步

**手动调整：**
- 分离音频
- 拖动对齐
- 微调位置

## 常见问题

### 问题 1：不同步

**原因：** 时间计算错误
**解决：** 重新对齐波形

### 问题 2：音量不平衡

**原因：** 音量设置不当
**解决：** 重新调整音量

### 问题 3：音效突兀

**原因：** 缺少过渡
**解决：** 添加淡入淡出
