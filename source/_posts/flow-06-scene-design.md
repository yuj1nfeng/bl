## 场景设计的核心价值

> "好的场景会讲故事。"

优秀的场景设计能够：

- **建立世界观** - 交代故事发生的时空背景
- **烘托情绪** - 用环境强化情感表达
- **引导视线** - 引导观众关注重点
- **提升质感** - 丰富画面细节和层次

<!-- more -->

## 一、场景设计流程

### 完整设计流程

```
剧本分析 → 参考收集 → 概念草图 → 方案确定 → 精细绘制 → 光影设定
   ↓          ↓          ↓          ↓          ↓          ↓
 1 天       1-2 天     2-3 天     2-3 天     3-5 天    1-2 天
```

### 场景分类

**按功能分类：**
```
主要场景（反复出现）:
├── 主角家/房间
├── 学校/公司
├── 常去场所（咖啡厅、公园）
└── 核心剧情发生地

次要场景（偶尔出现）:
├── 路过场所
├── 一次性剧情场景
└── 过渡场景

特殊场景（关键剧情）:
├── 高潮场景
├── 回忆场景
└── 幻想场景
```

**按空间分类：**
```
内景:
├── 房间内部
├── 建筑内部
└── 封闭空间

外景:
├── 街道
├── 自然景观
└── 开放空间
```

## 四、概念草图

### 草图绘制要点

**快速探索：**
```
目标：5-10 个不同构图
时间：每个 10-15 分钟
要点:
✓ 尝试不同视角
✓ 尝试不同布局
✓ 尝试不同空间关系
```

### 构图变化维度

**视角变化：**
```
视角探索:
├── 平视（常规视角）
├── 俯视（展示全貌）
├── 仰视（强调高度）
├── 鸟瞰（宏观视角）
└── 虫瞰（特殊视角）
```

**布局变化：**
```
布局探索:
├── 对称布局（稳定、正式）
├── 不对称布局（活泼、自然）
├── 中心聚焦（突出主体）
└── 分散布局（丰富、自然）
```

### 草图评估标准

```
评估维度（1-5 分）:
□ 功能性 - 满足剧情需要吗？
□ 美观度 - 视觉上好看吗？
□ 可行性 - 便于制作吗？
□ 独特性 - 有记忆点吗？
□ 氛围感 - 情绪准确吗？

总分≥20 分 → 进入深化
```

## 五、场景元素设计

### 空间结构

**空间三要素：**
```
1. 地面
   - 材质（木地板、瓷砖、草地）
   - 图案（纯色、花纹、拼花）
   - 状态（干净、杂乱）

2. 墙面
   - 材质（涂料、壁纸、砖墙）
   - 颜色（主色调）
   - 装饰（挂画、照片、架子）

3. 天花板
   - 高度（压抑/开阔）
   - 结构（平顶、斜顶、穹顶）
   - 灯具（吊灯、筒灯、灯带）
```

### 道具设计

**道具分类：**
```
功能性道具:
├── 家具（桌椅、床、柜子）
├── 电器（电视、电脑、灯具）
└── 日用品（餐具、书籍、装饰）

叙事性道具:
├── 关键道具（推动剧情）
├── 角色物品（体现性格）
└── 象征物品（隐喻意义）

氛围性道具:
├── 植物（绿植、花卉）
├── 装饰（挂画、摆件）
└── 纺织品（窗帘、地毯、靠垫）
```

**道具设计要点：**
```
✓ 符合时代背景
✓ 符合角色身份
✓ 风格统一
✓ 细节丰富但不过度
```

### 色彩设计

**场景配色原则：**
```
主色调（60%）:
- 决定场景整体印象
- 通常来自墙面、地面

辅助色（30%）:
- 丰富层次
- 来自家具、大型道具

点缀色（10%）:
- 画龙点睛
- 来自装饰品、小物件
```

**色彩与情绪：**
| 情绪 | 主色推荐 | 配色建议 |
|------|----------|----------|
| 温馨 | 米色、浅黄 | 暖棕、粉色 |
| 冷静 | 浅蓝、灰蓝 | 白色、银色 |
| 神秘 | 深紫、深蓝 | 黑色、金色 |
| 活力 | 橙色、黄色 | 绿色、红色 |
| 忧郁 | 灰色、蓝灰 | 深蓝、紫色 |

## 六、光影设计

### 光源类型

**自然光：**
```
特点:
- 来自太阳
- 随时间变化
- 方向性强

应用:
- 窗户射入的光线
- 清晨/黄昏的暖光
- 正午的强光
```

**人造光：**
```
特点:
- 来自灯具
- 可控性强
- 色温多样

应用:
- 吊灯、台灯
- 灯带、射灯
- 霓虹灯
```

**特殊光：**
```
类型:
- 魔法光效
- 屏幕光
- 反射光
- 透射光
```

### 光影效果

**光影作用：**
```
✓ 塑造体积感
✓ 营造氛围
✓ 引导视线
✓ 创造层次
```

**光影技法：**
```
1. 明暗对比
   - 强化戏剧性
   - 突出主体

2. 体积光
   - 丁达尔效应
   - 营造氛围

3. 反射光
   - 增加真实感
   - 丰富色彩

4. 投影
   - 增强空间感
   - 丰富画面
```

### 不同时间的用光

**清晨：**
```
特点:
- 色温偏冷（蓝调）
- 光线柔和
- 影子较长

应用:
- 新的开始
- 希望、清新
```

**正午：**
```
特点:
- 色温中性
- 光线强烈
- 影子短而清晰

应用:
- 日常场景
- 明快情绪
```

**黄昏：**
```
特点:
- 色温偏暖（橙红）
- 光线柔和
- 影子长

应用:
- 浪漫场景
- 离别、思念
```

**夜晚：**
```
特点:
- 整体偏暗
- 局部光源
- 强对比

应用:
- 神秘场景
- 紧张、危险
```

### 绘制流程

```
线稿 → 底色 → 阴影 → 高光 → 细节 → 调整
```

**推荐分层结构：**
```
场景文件分层:
├── 调整层（最上层）
│   ├── 色彩调整
│   └── 效果层
├── 特效层
│   ├── 光效
│   └── 粒子
├── 前景层
│   └── 近处道具
├── 角色层（预留）
├── 中景层
│   ├── 主要家具
│   └── 主要道具
├── 背景层
│   ├── 墙面
│   ├── 地面
│   └── 天花板
└── 线稿层（最下层）
```

## 八、氛围营造技巧

### 1. 色彩统一

**色调统一：**
```
方法:
1. 确定主色调
2. 所有元素向主色调靠拢
3. 用色彩调整层统一整体
```

### 2. 空气透视

**远近处理：**
```
原理:
- 近处：清晰、对比强、色彩饱和
- 远处：模糊、对比弱、色彩偏灰

应用:
- 增强空间纵深感
- 引导视觉焦点
```

### 3. 细节密度

**疏密对比：**
```
原则:
- 视觉中心：细节丰富
- 边缘区域：适当留白
- 避免平均用力
```

### 4. 故事性元素

**添加叙事细节：**
```
示例:
- 墙上的照片（暗示关系）
- 桌上的物品（体现习惯）
- 散落的物件（讲述事件）
```


## 常见场景类型示例

### 温馨卧室

```
设计要点:
- 色彩：暖色调（米色、粉色）
- 光线：柔和的台灯/窗光
- 道具：玩偶、照片、书籍
- 细节：褶皱的被子、散落的物品
```

### 神秘咖啡厅

```
设计要点:
- 色彩：深色调（棕色、暗红）
- 光线：局部光源、强对比
- 道具：复古灯具、老式家具
- 细节：蒸汽、光影斑驳
```

### 现代教室

```
设计要点:
- 色彩：明亮（白色、浅蓝）
- 光线：均匀的日光
- 道具：课桌椅、黑板、窗户
- 细节：阳光、窗帘飘动
```

### 奇幻森林

```
设计要点:
- 色彩：神秘（深绿、紫色）
- 光线：透过树叶的光束
- 道具：古树、藤蔓、蘑菇
- 细节：光斑、雾气、萤火虫
```

## 总结

场景设计是营造故事氛围的关键。关键要点：

1. **理解需求** - 场景服务于剧情
2. **收集参考** - 站在巨人肩膀上
3. **注重构图** - 好的构图成功一半
4. **光影营造** - 光是场景的灵魂
5. **细节叙事** - 让场景会讲故事
