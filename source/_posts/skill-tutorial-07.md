## 色彩情绪基础

### 色彩与情绪

**暖色系情绪：**
- 红色：热情、愤怒、危险
- 橙色：活力、温暖、友好
- 黄色：快乐、希望、警示

**冷色系情绪：**
- 蓝色：冷静、忧郁、神秘
- 绿色：自然、治愈、嫉妒
- 紫色：高贵、神秘、悲伤

**中性色情绪：**
- 白色：纯洁、空白、简约
- 黑色：神秘、死亡、高级
- 灰色：中性、沉稳、低调

### 色彩三要素与情绪

**色相：**
- 决定基本情绪倾向
- 暖色积极，冷色消极

**饱和度：**
- 高饱和：强烈情绪
- 低饱和：平静情绪

**明度：**
- 高明度：轻松愉快
- 低明度：沉重压抑

## 场景色彩设计

### 温馨场景

**色彩特点：**
- 暖色调为主
- 中高明度
- 中低对比

**配色方案：**
```
主色：米色、浅黄
辅色：粉色、浅橙
点缀：棕色
```

### 悲伤场景

**色彩特点：**
- 冷色调为主
- 低明度
- 低饱和

**配色方案：**
```
主色：蓝灰、灰紫
辅色：深灰
点缀：少量冷色
```

### 紧张场景

**色彩特点：**
- 高对比
- 冷暖对比
- 深色背景

**配色方案：**
```
主色：深蓝、黑色
辅色：红色（警示）
点缀：高亮色
```

### 浪漫场景

**色彩特点：**
- 粉紫色调
- 柔和过渡
- 中明度

**配色方案：**
```
主色：粉色、淡紫
辅色：浅蓝
点缀：金色光点
```

## 色彩转换技巧

### 情绪转换

**从开心到悲伤：**
```
前期：明亮暖色
转换：逐渐变冷
后期：阴暗冷色
```

**从压抑到释放：**
```
前期：低明度冷色
转换：光线射入
后期：明亮暖色
```

### 时间转换

**白天到夜晚：**
```
白天：高明度暖色
黄昏：橙红色调
夜晚：低明度冷色
```

**季节转换：**
```
春：嫩绿、粉色
夏：深绿、蓝色
秋：金黄、橙红
冬：白色、蓝灰
```

## 色彩统一技巧

### 色调统一

**方法：**
1. 确定主色调
2. 所有颜色向主色调靠拢
3. 使用色彩调整层

**效果：**
- 画面和谐
- 情绪统一
- 风格鲜明

### 色彩呼应

**方法：**
- 重要颜色重复出现
- 角色代表色呼应
- 关键道具颜色呼应

**作用：**
- 增强联系
- 强化记忆
- 深化主题

## 特殊色彩效果

### 回忆场景

**色彩特点：**
- 降低饱和
- 添加柔光
- 偏黄绿色调

**处理方法：**
- 降低饱和度 30%
- 添加柔光效果
- 色温偏暖

### 幻想场景

**色彩特点：**
- 高饱和
- 非现实色彩
- 梦幻感

**处理方法：**
- 提高饱和度
- 添加光晕
- 使用特殊色调

### 恐怖场景

**色彩特点：**
- 低明度
- 绿色调
- 强对比

**处理方法：**
- 降低整体亮度
- 色调偏绿
- 增强对比度

## 常见问题

### 问题 1：色彩太乱

**症状：** 颜色太多，没有主色调

**解决：**
- 限制颜色数量
- 确定主色调
- 统一色彩倾向

### 问题 2：色彩太灰

**症状：** 画面发灰，没有精神

**解决：**
- 提高饱和度
- 增强对比
- 添加纯色点缀

### 问题 3：色彩与情绪不符

**症状：** 悲伤场景用暖色

**解决：**
- 明确场景情绪
- 选择对应色彩
- 参考色彩心理学
