## 开场设计

### 开场要素

**标准结构（2 分钟）：**
```
0:00-0:15  片头音乐
0:15-0:45  节目介绍
0:45-1:30  本期主题引入
1:30-2:00  嘉宾介绍
```

### 开场技巧

**钩子开场：**
- 抛出一个问题
- 讲一个故事
- 引用一句金句
- 播放一段精彩片段

**示例：**
```
"你有没有想过，那些让你哭得稀里哗啦的漫剧角色，
背后的配音演员可能是一群逗比？
今天我们就来揭秘..."
```

### 片头音乐

**选择要点：**
- 符合节目调性
- 时长 15-30 秒
- 有辨识度
- 版权清晰

## 结尾设计

### 结尾要素

**标准结构（3 分钟）：**
```
0:00-1:00  内容总结
1:00-2:00  互动引导
2:00-3:00  下期预告 + 片尾音乐
```

### 结尾技巧

**总结要点：**
- 提炼 3 个核心观点
- 输出金句
- 行动号召

**互动引导：**
- 提问引发讨论
- 引导评论
- 邀请订阅

**示例：**
```
"今天的节目就到这里，
如果你有什么想聊的，
欢迎在评论区留言。
下期我们会聊...
别忘了订阅我们！"
```

### 片尾音乐

**选择要点：**
- 与片头呼应
- 渐弱结束
- 时长 30-60 秒

## 固定 Slogan

**作用：**
- 增强辨识度
- 强化品牌
- 形成记忆点

**示例：**
- "这里是 XXX，我们下期见"
- "用声音记录漫剧故事"
