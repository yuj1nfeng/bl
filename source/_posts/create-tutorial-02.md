## 经典故事结构

### 三幕式结构

**第一幕：铺垫（25%）**
- 介绍主角和世界观
- 展示日常生活
- 引发事件出现
- 主角做出决定

**第二幕：对抗（50%）**
- 面对挑战和阻碍
- 学习和成长
- 中点转折
- 最低谷时刻

**第三幕：解决（25%）**
- 最终对决
- 高潮爆发
- 问题解决
- 新的平衡

### 英雄之旅

**12 个阶段：**
1. 平凡世界
2. 冒险召唤
3. 拒绝召唤
4. 遇见导师
5. 跨越门槛
6. 考验盟友敌人
7. 接近洞穴
8. 严峻考验
9. 获得奖励
10. 回归之路
11. 复活
12. 满载而归

### 起承转合

**起（引入）：**
- 交代背景
- 介绍人物
- 设定情境

**承（发展）：**
- 情节推进
- 矛盾积累
- 关系发展

**转（转折）：**
- 意外事件
- 情节反转
- 情绪高潮

**合（结局）：**
- 问题解决
- 主题升华
- 留下余味

## 情节设计技巧

### 冲突设置

**冲突类型：**
- 人与人（外部冲突）
- 人与环境
- 人与自己（内心冲突）

**冲突升级：**
```
小冲突 → 中冲突 → 大冲突 → 最终冲突
```

### 悬念设计

**悬念类型：**
- 结果悬念（会发生什么）
- 时间悬念（来得及吗）
- 身份悬念（他是谁）
- 动机悬念（为什么）

**悬念设置技巧：**
- 提前暗示
- 延迟揭示
- 误导观众
- 意外反转

### 节奏控制

**节奏变化：**
```
缓慢 → 加快 → 缓慢 → 快速 → 高潮
```

**张弛有度：**
- 紧张后放松
- 高潮后平静
- 给观众呼吸空间

## 多集结构设计

### 单集结构

**标准结构（3 分钟）：**
```
0:00-0:15  开场钩子
0:15-1:00  情境建立
1:00-2:00  冲突发展
2:00-2:30  高潮转折
2:30-3:00  结尾悬念
```

### 季结构设计

**12 集结构：**
```
1-3 集：引入阶段
4-6 集：发展阶段
7-9 集：转折阶段
10-12 集：高潮结局
```

**每集功能：**
- 第 1 集：引入世界观
- 第 3 集：第一个高潮
- 第 6 集：中点转折
- 第 9 集：最低谷
- 第 12 集：最终高潮
