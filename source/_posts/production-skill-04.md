## 手部结构基础

### 手部比例

**基本比例：**
```
手掌长度：约等于脸长
手指长度：中指最长
手掌宽度：约等于手掌长度
```

**手指比例：**
```
中指：最长
食指 + 无名指：次之
小指：最短（到无名指第二关节）
拇指：最短（但最粗）
```

### 手部结构

**手掌部分：**
- 掌心
- 手背
- 大鱼际（拇指根部）
- 小鱼际（小指根部）

**手指部分：**
- 指根（与手掌连接）
- 第一指节
- 第二指节
- 指尖

**关节：**
- 指根关节
- 第一指关节
- 第二指关节

## 常见手势

### 握拳手势

**特点：**
- 四指弯曲
- 拇指压住食指
- 力量感强

**绘制要点：**
```
1. 画出手掌方块
2. 四指弯曲向掌心
3. 拇指跨过食指
4. 注意指关节凸起
```

### 张开手势

**特点：**
- 五指展开
- 手掌打开
- 表达性强

**绘制要点：**
```
1. 画出手掌形状
2. 五指放射状展开
3. 注意手指长度差异
4. 掌心微微凹陷
```

### 指点手势

**特点：**
- 食指伸出
- 其他手指弯曲
- 指示方向

**绘制要点：**
```
1. 食指伸直
2. 中指、无名指、小指弯曲
3. 拇指自然弯曲
4. 注意食指力度
```

### OK 手势

**特点：**
- 拇指和食指成圈
- 其他三指伸直
- 表示肯定

**绘制要点：**
```
1. 拇指食指指尖相接
2. 形成圆形
3. 其他三指自然伸展
4. 注意手部角度
```

### 胜利手势

**特点：**
- 食指和中指伸直
- 其他手指弯曲
- V 字形状

**绘制要点：**
```
1. 食指中指伸直分开
2. 无名指小指弯曲
3. 拇指压住弯曲手指
4. V 字角度适中
```

### 抓取手势

**特点：**
- 五指弯曲
- 向中心聚拢
- 抓取动作

**绘制要点：**
```
1. 手掌微弓
2. 五指弯曲向中心
3. 指尖不接触
4. 注意透视
```

## 不同角度手部

### 正面手部

**要点：**
- 五指清晰可见
- 注意长度差异
- 掌心细节

### 侧面手部

**要点：**
- 手掌厚度明显
- 手指重叠
- 注意透视

### 背面手部

**要点：**
- 手背静脉可见
- 指关节凸起
- 指甲可见

### 俯视手部

**要点：**
- 手指变短
- 手掌面积大
- 指尖可见

### 仰视手部

**要点：**
- 掌心面积大
- 手指上翘
- 指甲不可见

## 手部绘制步骤

### Step 1: 画手掌基本形

```
1. 画一个方形或梯形
2. 确定手掌方向
3. 标记手腕位置
```

### Step 2: 确定手指位置

```
1. 画手指中心线
2. 标记指根位置
3. 确定手指长度
```

### Step 3: 画手指轮廓

```
1. 从指根画起
2. 注意粗细变化
3. 指尖圆润
```

### Step 4: 添加关节细节

```
1. 标记关节位置
2. 画出关节凸起
3. 添加皱纹细节
```

### Step 5: 完善轮廓

```
1. 平滑线条
2. 添加厚度
3. 注意指甲
```

### Step 6: 上色

```
1. 基础肤色
2. 关节处加深
3. 指甲上色
4. 添加阴影
```

## 手部动态

### 写字手势

**要点：**
- 拇指食指捏笔
- 中指支撑
- 手腕灵活

### 打电话手势

**要点：**
- 拇指小指伸展
- 其他三指弯曲
- 贴近耳朵

### 托腮手势

**要点：**
- 手掌托住下巴
- 手指自然弯曲
- 手肘支撑

### 拍手手势

**要点：**
- 双手相对
- 手指伸直
- 动态模糊

## 常见问题

### 问题 1：手指像香肠

**原因：** 太粗太圆，没有关节
**解决：** 添加关节，注意粗细变化

### 问题 2：手部太僵硬

**原因：** 手指太直，没有自然弯曲
**解决：** 添加自然弯曲，放松姿势

### 问题 3：比例不对

**原因：** 手指长度错误
**解决：** 记住比例，多参考

### 问题 4：透视错误

**原因：** 不理解手部体积
**解决：** 理解手部结构，多练习
