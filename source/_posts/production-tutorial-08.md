## 口型基础

### 基础口型

**8 个基础口型：**

| 口型 | 发音 | 嘴型 |
|------|------|------|
| A | m, b, p | 闭合 |
| B | e, i, ü | 微张 |
| C | a, ä | 中等 |
| D | o, u | 大张 |
| E | ei, ai | 咧嘴 |
| F | f, v | 咬唇 |
| L | l, n | 舌抵上颚 |
| W | w, q | 圆唇 |

### 口型转换

**转换原则：**
- 自然过渡
- 避免跳跃
- 保持流畅

**转换时间：**
- 快速转换：2-3 帧
- 正常转换：4-6 帧
- 慢速转换：6-8 帧

## 制作流程

### Step 1: 音频分析

**分析内容：**
- 台词内容
- 发音节奏
- 重音位置
- 停顿位置

**标记方法：**
```
在时间轴上标记:
- 每个音节开始
- 重音位置
- 停顿位置
```

### Step 2: 口型规划

**规划原则：**
- 提前 2-3 帧（声音传播延迟）
- 重点音节强调
- 句尾自然闭合

**口型分配：**
```
示例台词："你好吗"

你 (nǐ): L 口型 → B 口型
好 (hǎo): C 口型 → D 口型
吗 (ma): A 口型 → B 口型
```

### Step 3: 口型绘制

**绘制要点：**
- 符合角色特征
- 表情配合
- 自然过渡

**分层绘制：**
```
上层：上嘴唇
中层：下嘴唇
下层：舌头/牙齿
```

### Step 4: 同步调整

**检查内容：**
- 音画同步
- 过渡自然
- 表情配合

**调整方法：**
- 微调时间
- 修改口型
- 添加过渡

## 口型技巧

### 简化技巧

**2 帧一切：**
- 不需要每帧都变
- 2-3 帧切换一次
- 节省工作量

**关键口型：**
- 只绘制关键口型
- 中间自动过渡
- 适合简单对话

### 真实感技巧

**呼吸配合：**
- 说话前吸气
- 句间换气
- 情绪呼吸

**表情配合：**
- 眉毛动作
- 眼神变化
- 脸部肌肉

**细节添加：**
- 舌头动作
- 牙齿显示
- 唾液反光

## 常见问题

### 问题 1：不同步

**原因：**
- 时间计算错误
- 帧率不匹配

**解决：**
- 重新计算
- 调整时间轴

### 问题 2：太机械

**原因：**
- 口型太标准
- 缺少变化

**解决：**
- 添加微变化
- 加入表情

### 问题 3：工作量大

**解决：**
- 建立口型库
- 复用口型
- 使用插件
