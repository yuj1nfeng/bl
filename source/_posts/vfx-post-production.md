## 后期制作的重要性

> "前期决定下限，后期决定上限"

后期制作是将原始素材转化为最终成品的关键环节。优秀的后期可以：

- **提升视觉品质** - 通过调色、合成让画面更加精致
- **增强情感表达** - 利用特效强化情绪渲染
- **弥补前期不足** - 修复拍摄/绘制中的缺陷
- **统一视觉风格** - 确保全片视觉一致性

## 一、素材整理与管理

### 文件夹结构规范

```
Project_Name/
├── 01_Footage/          # 原始素材
│   ├── Animation/       # 动画序列
│   ├── Background/      # 背景图
│   └── Reference/       # 参考资料
├── 02_Audio/            # 音频文件
│   ├── BGM/            # 背景音乐
│   ├── SFX/            # 音效
│   └── Voice/          # 配音
├── 03_Project/          # 工程文件
├── 04_Exports/          # 输出文件
│   ├── Draft/          # 草稿版本
│   └── Final/          # 最终版本
└── 05_Docs/            # 文档资料
```

### 命名规范

- 使用英文或拼音，避免中文路径问题
- 包含版本号和日期：`SC01_v02_20260303.aep`
- 统一日期格式：YYYYMMDD

## 二、视觉特效技术

### 1. 粒子效果

粒子系统用于创建：

- 魔法光效
- 火焰、烟雾
- 雨雪天气
- 星光、尘埃

**After Effects 粒子插件推荐：**
- Trapcode Particular（行业标准）
- Form（抽象图形）
- Stardust（节点式粒子）

### 2. 光效处理

**常见光效类型：**

| 效果 | 用途 | 实现方法 |
|------|------|----------|
| 辉光 | 魔法、能量 | Glow + 自定义光晕 |
| 光斑 | 镜头光晕 | Optical Flares |
| 光线 | 体积光 | God Rays 插件 |
| 闪烁 | 能量波动 | 关键帧动画 |

### 3. 动态图形

**MG 动画元素：**

- 转场动画
- 信息图表
- UI 界面动效
- 文字动画

### 4. 合成技巧

**多层合成原则：**

```
从上到下：
1. 调色层（Adjustment Layer）
2. 特效层（VFX）
3. 前景层（Foreground）
4. 角色层（Character）
5. 背景层（Background）
```

**混合模式应用：**

- **Screen/滤色** - 去黑留亮，用于光效
- **Multiply/正片叠底** - 去白留黑，用于阴影
- **Add/相加** - 增强亮度
- **Overlay/叠加** - 增强对比

## 三、色彩校正与调色

### 校正 vs 调色

**色彩校正（Correction）**
- 修复曝光问题
- 统一不同镜头的色温
- 还原真实色彩

**色彩分级（Grading）**
- 创造特定氛围
- 建立视觉风格
- 强化情感表达

### 调色流程

```
1. 一级调色（Primary）
   └─→ 调整整体曝光、对比度、白平衡
   
2. 二级调色（Secondary）
   └─→ 局部调整、肤色校正、选择性调色
   
3. 风格化（Look）
   └─→ 应用 LUT、创建独特色调
```

### 常用调色工具

**After Effects / Premiere:**
- Lumetri Color
- Curves（曲线）
- Levels（色阶）

**DaVinci Resolve（专业调色）:**
- 节点式调色
- 强大的二级调色
- 优秀的肤色处理

### 色彩情绪参考

| 情绪 | 色温 | 饱和度 | 对比度 |
|------|------|--------|--------|
| 温馨 | 暖 | 中高 | 中 |
| 悲伤 | 冷 | 低 | 低 |
| 紧张 | 冷 | 中 | 高 |
| 欢快 | 暖 | 高 | 中高 |
| 神秘 | 冷 | 低 | 中 |

## 四、转场设计

### 基础转场

- **硬切（Cut）** - 最常用，干净利落
- **淡入淡出（Fade）** - 场景转换、时间流逝
- **溶解（Dissolve）** - 柔和过渡

### 创意转场

**匹配剪辑（Match Cut）**
- 利用形状、颜色、运动的相似性转场

**遮挡转场（Mask Transition）**
- 利用前景物体遮挡完成转场

**动态转场（Motion Transition）**
- 利用摄像机运动或物体运动转场

```
转场设计原则：
✓ 服务于叙事
✓ 保持节奏一致
✓ 避免过度使用
✗ 不要为了炫技而用
```

## 五、输出规范

### 网络平台标准

**Bilibili / YouTube:**
```
分辨率：1920x1080 (1080p) 或 3840x2160 (4K)
帧率：24/25/30/60 fps
码率：1080p 建议 6000-8000 kbps
格式：MP4 (H.264)
音频：AAC, 320 kbps, 48kHz
```

### 编码设置建议

**H.264 编码参数：**
- Profile: High
- Level: 5.1
- Keyframe: 2s 间隔
- VBR 2 次编码
