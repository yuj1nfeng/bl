## 表情基础

### 基本表情类型

**七种基本表情：**
1. 开心
2. 悲伤
3. 愤怒
4. 惊讶
5. 恐惧
6. 厌恶
7. 轻蔑

### 表情三要素

**眼睛：** 心灵的窗户
**嘴巴：** 情绪的直接表达
**眉毛：** 情感的强化

## 表情绘制

### 开心表情

**特征：**
- 嘴角上扬
- 眼睛弯曲
- 眉毛舒展

**绘制要点：**
```
眼睛：下眼睑向上弯曲
嘴巴：嘴角向上，露齿或不露齿
脸颊：可添加红晕
```

### 悲伤表情

**特征：**
- 嘴角下垂
- 眼角下垂
- 眉头紧锁

**绘制要点：**
```
眼睛：上眼睑下垂，眼神黯淡
嘴巴：嘴角向下
眉毛：内眼角向上
```

### 愤怒表情

**特征：**
- 眉头紧锁
- 嘴角紧绷
- 眼神锐利

**绘制要点：**
```
眉毛：向下压，呈 V 字形
眼睛：瞪大，瞳孔缩小
嘴巴：紧闭或张开露齿
```

### 惊讶表情

**特征：**
- 眼睛睁大
- 嘴巴张开
- 眉毛上扬

**绘制要点：**
```
眼睛：瞳孔缩小，眼白增多
嘴巴：自然张开呈 O 型
眉毛：高高扬起
```

## 表情变化

### 表情过渡

**渐变原则：**
- 起始表情
- 过渡帧
- 目标表情

**时间控制：**
- 快速变化：3-5 帧
- 正常变化：6-12 帧
- 慢速变化：12-24 帧

### 微表情

**定义：** 短暂的面部表情变化
**时长：** 1/25 秒 -1/5 秒
**作用：** 增加真实感
