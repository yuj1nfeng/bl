## 调色的核心价值

> "调色是情绪的艺术。"

优秀的调色能够：

- **统一画面** - 确保镜头间色彩一致

## 一、调色基础

### 色彩理论

**色彩三要素：**
```
色相（Hue）:
- 颜色的种类
- 红、橙、黄、绿、青、蓝、紫

饱和度（Saturation）:
- 颜色的纯度
- 高饱和 = 鲜艳
- 低饱和 = 灰暗

明度（Value/Lightness）:
- 颜色的明暗
- 高明度 = 明亮
- 低明度 = 暗沉
```

**色轮关系：**
```
互补色:
- 色轮相对位置
- 红 - 青、绿 - 品红、蓝 - 黄
- 对比强烈

类似色:
- 色轮相邻位置
- 和谐统一

三色组合:
- 等边三角形
- 丰富平衡
```

**色温：**
```
暖色（3000K-4500K）:
- 红、橙、黄
- 温馨、热情、活力

冷色（5000K-7000K）:
- 蓝、青
- 冷静、忧郁、神秘

中性（4500K-5000K）:
- 自然光
- 平衡、真实
```

## 二、调色流程

### 标准流程

```
素材分析 → 一级调色 → 二级调色 → 风格化 → 统一检查 → 输出
```

### 一级调色（Primary Correction）

**目标：**
```
✓ 校正曝光
✓ 调整对比度
✓ 校正白平衡
✓ 统一基础色彩
```

**调整顺序：**
```
1. 曝光调整
   - 确保画面亮度正确
   - 不过曝不欠曝

2. 对比度调整
   - 建立明暗关系
   - 增强画面层次

3. 白平衡校正
   - 校正色温
   - 还原真实色彩

4. 饱和度调整
   - 调整色彩鲜艳度
   - 避免过饱和
```

**示波器使用：**
```
波形图（Waveform）:
- 显示亮度分布
- 0-100 IRE 范围
- 避免超出范围

矢量图（Vectorscope）:
- 显示色彩分布
- 肤色线参考
- 色彩平衡检查

直方图（Histogram）:
- 显示像素分布
- 检查曝光
- 避免剪切
```

### 二级调色（Secondary Correction）

**目标：**
```
✓ 局部调整
✓ 选择性调色
✓ 肤色校正
✓ 重点突出
```

**常用工具：**
```
HSL 限定:
- Hue（色相）
- Saturation（饱和度）
- Luminance（明度）
- 选择特定颜色范围

蒙版:
- 形状蒙版
- 渐变蒙版
- 跟踪蒙版

窗口:
- 圆形窗口
- 方形窗口
- 自定义形状
```

**调整内容：**
```
肤色校正:
- 确保肤色自然
- 不同镜头统一
- 符合人物特点

重点突出:
- 提亮主体
- 压暗背景
- 引导视线

色彩分离:
- 高光偏暖
- 阴影偏冷
- 增强层次
```

### 风格化（Look Development）

**目标：**
```
✓ 建立视觉风格
✓ 强化情绪表达
✓ 形成作品特色
```

**常见风格：**

**清新明亮：**
```
特点:
- 高亮度
- 低对比
- 偏冷色调
- 低饱和

参数:
- 曝光：+0.3 至 +0.7
- 对比度：-10% 至 -20%
- 色温：偏冷（5500K-6500K）
- 饱和度：-10% 至 -20%
```

**电影感：**
```
特点:
- 高对比
- 压暗四周
- 青橙色调
- 添加颗粒

参数:
- 对比度：+20% 至 +30%
- 暗部：提升黑位
- 高光：偏青
- 阴影：偏橙
- 颗粒：5-10%
```

**复古风格：**
```
特点:
- 降低饱和
- 偏黄绿色调
- 柔化效果
- 添加颗粒

参数:
- 饱和度：-30% 至 -40%
- 色温：偏暖（4000K-5000K）
- 色调：偏绿
- 柔光：10-20%
```

**赛博朋克：**
```
特点:
- 高饱和
- 蓝紫主调
- 霓虹色彩
- 强对比

参数:
- 饱和度：+30% 至 +50%
- 蓝色：增强
- 紫色：增强
- 对比度：+30%
```

## 四、LUT 应用

### LUT 基础

**什么是 LUT：**
```
LUT（Look Up Table）:
- 色彩查找表
- 输入值→输出值映射
- 快速应用色彩风格
```

**LUT 类型：**
```
技术 LUT:
- 色彩空间转换
- Log 转 Rec.709
- 相机 LUT

创意 LUT:
- 风格化预设
- 电影 LUT
- 自定义 LUT
```

### LUT 使用

**应用方法：**
```
1. 一级调色完成后应用
2. 调整 LUT 强度（0-100%）
3. 微调匹配画面
```

**注意事项：**
```
✓ LUT 是起点不是终点
✓ 需要根据画面调整
✓ 不要过度依赖
✓ 注意色彩准确性
```

## 六、常见问题

### 问题 1：色彩断层

**症状：** 渐变区域出现色带

**解决：**
- 使用 10bit 或更高采样
- 添加轻微噪点
- 避免过度调整
- 使用抖动

### 问题 2：肤色不自然

**症状：** 人物肤色怪异

**解决：**
- 单独调整肤色
- 参考肤色线
- 降低饱和度
- 检查白平衡

### 问题 3：画面发灰

**症状：** 对比度不足，画面发灰

**解决：**
- 增加对比度
- 调整 S 曲线
- 分离高光阴影
- 检查曝光

### 问题 4：色彩溢出

**症状：** 色彩过于饱和，细节丢失

**解决：**
- 降低饱和度
- 检查示波器
- 限制色彩范围
- 使用柔光混合
