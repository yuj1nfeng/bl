## 配音的重要性

> "好的配音能让角色活过来。"

优秀的配音能够：

- **塑造角色性格** - 声音是角色的第二张脸
- **传递情感** - 用声音感染观众
- **提升作品质感** - 专业配音提升整体品质
- **增强代入感** - 让观众沉浸其中

## 一、配音录制流程

### 完整流程

```
剧本标注 → 声优选拔 → 录音准备 → 正式录制 → 补录检查 → 音频整理
   ↓          ↓          ↓          ↓          ↓          ↓
 1-2 天     2-3 天     1 天       2-4 小时    1-2 小时   1 天
```

### 录制方式

**棚录（推荐）**
```
优点:
✓ 音质好
✓ 环境可控
✓ 专业设备
✓ 导演可现场指导

缺点:
✗ 成本高
✗ 需要预约
```

**远程录制**
```
优点:
✓ 成本低
✓ 时间灵活
✓ 地域不限

缺点:
✗ 音质参差不齐
✗ 沟通成本高
✗ 无法实时指导
```

**自制录音**
```
优点:
✓ 成本最低
✓ 时间自由

缺点:
✗ 音质难保证
✗ 环境噪音
✗ 设备限制

适用：Demo、临时配音
```

## 二、剧本标注

### 标注内容

**情绪标注：**
```
示例:
林小满
（紧张地）
请问，这里是「时光咖啡厅」吗？

陈默
（神秘地微笑）
现在是，但十分钟后不是。
```

**语气标注：**
```
常见语气:
- 轻声/大声
- 快速/慢速
- 颤抖/坚定
- 犹豫/果断
- 温柔/冷漠
```

**气息标注：**
```
气息类型:
- 深呼吸
- 叹气
- 喘息
- 抽泣
- 轻笑
```

### 完整标注示例

```
【SC.001】内景 咖啡厅 日

林小满
（深吸一口气，紧张）
请问...（停顿）这里是「时光咖啡厅」吗？
（声音微颤）

陈默
（擦拭杯子，漫不经心）
现在是，但十分钟后不是。
（轻笑）

林小满
（惊讶，上前一步）
你什么意思？

陈默
（放下杯子，认真）
改变过去，是要付出代价的。
（低沉）
```

## 三、声优选拔

### 选拔流程

```
发布招募 → 收集试音 → 初步筛选 → 复选 → 确定人选
```

### 试音材料准备

**试音稿：**
```
试音稿要求:
- 包含角色典型台词
- 展示不同情绪
- 时长 1-2 分钟
- 提供角色设定
```

**试音稿示例：**
```markdown
# 林小满 试音稿

## 角色设定
- 年龄：20 岁
- 性格：内向敏感，善良执着
- 身份：大学生

## 试音片段

片段 1（紧张）:
"请问，这里是「时光咖啡厅」吗？"

片段 2（坚定）:
"我确定要这么做。"

片段 3（悲伤）:
"奶奶...我真的好想你。"

片段 4（开心）:
"太好了！我们成功了！"
```

### 选拔标准

```
评估维度:
├── 音色匹配度（40%）
├── 表演能力（30%）
├── 台词功底（20%）
└── 配合度（10%）
```

## 四、录音准备

### 录音棚选择

**专业录音棚：**
```
设备要求:
✓ 专业麦克风（Neumann、Sony）
✓ 声卡/调音台
✓ 监听耳机
✓ 隔音良好

费用参考:
- 一线城市：300-800 元/小时
- 二线城市：200-500 元/小时
```

### 录音前准备

**声优准备：**
```
✓ 熟读剧本
✓ 理解角色
✓ 保护嗓子（忌辛辣、冷饮）
✓ 充足休息
```

**导演准备：**
```
✓ 标注剧本
✓ 准备参考音频
✓ 准备饮用水
✓ 准备录音流程表
```

**技术准备：**
```
✓ 设备调试
✓ 电平测试
✓ 试录检查
✓ 备份方案
```

## 五、正式录制

### 录制流程

```
1. 热身开嗓（15 分钟）
   - 唇颤音练习
   - 音阶练习
   - 绕口令

2. 角色对词（30 分钟）
   - 熟悉台词
   - 找角色感觉
   - 调整状态

3. 正式录制（2-3 小时）
   - 按场景顺序
   - 每句录 3-5 遍
   - 实时监听

4. 补录检查（30 分钟）
   - 检查遗漏
   - 补录问题句
   - 确认质量
```

### 录音技巧

**麦克风使用：**
```
距离:
- 正常对话：15-20cm
- 轻声细语：10-15cm
- 大喊大叫：25-30cm

角度:
- 正对麦克风
- 避免正对喷口
- 侧 15 度减少喷麦
```

**防喷麦技巧：**
```
✓ 使用防喷罩
✓ 控制爆破音（b, p, t）
✓ 适当侧头
✓ 后期处理
```

### 导演指导技巧

**情绪引导：**
```
方法:
- 描述情境
- 类比参考
- 示范表演
- 回忆引导

示例:
"想象你等了很久的消息，终于收到了"
"就像你第一次见到喜欢的人那样"
```

**节奏把控：**
```
要点:
- 语速符合情绪
- 停顿自然
- 不要赶词
- 留足气息
```

**多版本录制：**
```
建议:
✓ 每句录 3-5 遍
✓ 不同情绪版本
✓ 不同节奏版本
✓ 备用选择
```

## 六、声音表演技巧

### 角色声音设计

**声音要素：**
```
音调:
- 高音：年轻、活泼、紧张
- 中音：正常、平和
- 低音：成熟、稳重、神秘

音色:
- 明亮：开朗、阳光
- 沙哑：沧桑、神秘
- 柔和：温柔、善良
- 尖锐：刻薄、紧张

语速:
- 快速：急躁、兴奋、紧张
- 正常：平和
- 慢速：稳重、犹豫、悲伤
```

### 情绪表达技巧

**开心：**
```
要点:
✓ 音调上扬
✓ 语速稍快
✓ 嘴角上扬（影响发声）
✓ 气息轻快
```

**悲伤：**
```
要点:
✓ 音调低沉
✓ 语速缓慢
✓ 气息颤抖
✓ 适当哽咽
```

**愤怒：**
```
要点:
✓ 音量增大
✓ 音调降低
✓ 语速加快
✓ 气息粗重
```

**紧张：**
```
要点:
✓ 音调偏高
✓ 语速不稳
✓ 气息短促
✓ 适当结巴
```

**惊讶：**
```
要点:
✓ 音调突然升高
✓ 音量突增
✓ 短促有力
✓ 倒吸一口气
```

### 特殊表演

**气声：**
```
用途：秘密、亲密、虚弱
技巧:
- 减少声带振动
- 增加气息比例
- 靠近麦克风
```

**耳语：**
```
用途：悄悄话、内心独白
技巧:
- 几乎不振动声带
- 清晰咬字
- 近距离录制
```

**哭腔：**
```
技巧:
- 喉咙收紧
- 气息颤抖
- 声音哽咽
- 真实眼泪（可选）
```

**笑声：**
```
类型:
- 微笑：轻笑，气息声
- 大笑：放开笑，注意气息
- 冷笑：鼻腔共鸣
- 苦笑：笑中带叹

要点:
✓ 自然不做作
✓ 符合角色性格
✓ 注意气息控制
```

## 七、音频后期

### 基础处理

**降噪：**
```
步骤:
1. 采集噪音样本
2. 应用降噪
3. 检查音质损失

要点:
✓ 不要过度降噪
✓ 保留自然空间感
```

**均衡（EQ）：**
```
常见调整:
- 低频（80-150Hz）：减少浑浊
- 中低频（200-500Hz）：减少闷感
- 中高频（2-5kHz）：增加清晰度
- 高频（10kHz+）：增加空气感
```

**压缩：**
```
作用:
- 平衡音量
- 增加饱满度

参数:
- 比率：2:1 - 4:1
- 启动：5-10ms
- 释放：50-100ms
```

### 高级处理

**混响：**
```
用途:
- 增加空间感
- 统一音色

参数:
- 房间大小
- 混响时间
- 干湿比
```

**音量标准化：**
```
标准:
- 对白：-6dB 峰值
- 响度：-23 LUFS（广播标准）
- 留有余量给 BGM/音效
```

## 八、常见问题

### 问题 1：喷麦严重

**解决：**
- 使用防喷罩
- 调整录音角度
- 后期高通滤波
- 重新录制

### 问题 2：环境噪音

**解决：**
- 选择安静环境
- 关闭电器设备
- 增加吸音材料
- 后期降噪

### 问题 3：表演生硬

**解决：**
- 充分理解角色
- 导演引导情绪
- 多录几个版本
- 参考优秀作品

### 问题 4：口齿不清

**解决：**
- 热身练习
- 放慢语速
- 夸张咬字
- 后期调整
