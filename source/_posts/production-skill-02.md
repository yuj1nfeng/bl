## 头发基础

### 头发生长规律

**生长方向：**
- 从发旋向外辐射
- 前额向下生长
- 两侧向后生长
- 后脑向下生长

**发旋位置：**
- 通常在头顶偏后
- 决定头发流向
- 影响发型设计

### 头发分组

**主要分组：**
```
前发：刘海部分
侧发：耳朵两侧
后发：后脑勺部分
顶发：头顶部分
```

## 发型设计

### 长发发型

**特点：**
- 长度过肩
- 变化多样
- 女性角色常用

**绘制要点：**
1. 注意重力影响
2. 发梢自然弯曲
3. 分组清晰
4. 添加飘动感

### 短发发型

**特点：**
- 长度在耳部附近
- 清爽利落
- 男女角色通用

**绘制要点：**
1. 贴合头部结构
2. 发梢方向明确
3. 层次分明
4. 注意厚度

### 马尾发型

**特点：**
- 头发束起
- 活力感强
- 运动角色常用

**绘制要点：**
1. 确定扎发位置
2. 头发流向扎点
3. 马尾自然下垂
4. 添加发饰

### 双马尾发型

**特点：**
- 两侧扎起
- 可爱活泼
- 少女角色常用

**绘制要点：**
1. 对称分布
2. 考虑重力
3. 发量分配均匀
4. 发饰点缀

### 卷发发型

**特点：**
- 波浪卷曲
- 浪漫优雅
- 成熟角色常用

**绘制要点：**
1. S 形曲线
2. 卷曲有规律
3. 注意体积感
4. 高光跟随卷曲

## 头发绘制步骤

### Step 1: 画头部轮廓

```
1. 画出头部基本形
2. 标记发旋位置
3. 确定发型类型
```

### Step 2: 画头发外轮廓

```
1. 确定头发厚度
2. 画出发型轮廓
3. 注意整体形状
```

### Step 3: 分组绘制

```
1. 分成几大组
2. 画出每组流向
3. 注意层次关系
```

### Step 4: 添加发丝细节

```
1. 在分组基础上
2. 添加发丝线条
3. 注意疏密变化
```

### Step 5: 画上高光

```
1. 确定光源
2. 添加条状高光
3. 跟随头发走向
```

### Step 6: 上色

```
1. 基础色
2. 阴影色
3. 高光色
4. 环境色
```

## 头发光影

### 光源方向

**顶光：**
- 头顶亮
- 两侧暗
- 发梢暗

**侧光：**
- 一侧亮
- 一侧暗
- 明暗对比强

**逆光：**
- 边缘亮
- 中间暗
- 轮廓光明显

### 高光处理

**高光形状：**
- 长条状（直发）
- 点状（卷发）
- 块状（短发）

**高光位置：**
- 头顶
- 发旋附近
- 头发弯曲处

## 不同风格头发

### 写实风格

**特点：**
- 发丝细节多
- 光影真实
- 质感强烈

**技巧：**
- 分层绘制
- 精细刻画
- 注意质感

### 卡通风格

**特点：**
- 简化发丝
- 色块分明
- 高光明显

**技巧：**
- 分组清晰
- 平涂上色
- 简化细节

### 厚涂风格

**特点：**
- 笔触感强
- 色彩丰富
- 体积感强

**技巧：**
- 块面塑造
- 色彩叠加
- 笔触方向

## 常见问题

### 问题 1：头发像头盔

**原因：** 太贴合头部，没有厚度
**解决：** 增加头发厚度，添加飘散发丝

### 问题 2：头发杂乱无章

**原因：** 没有分组，发丝方向混乱
**解决：** 先分组，再画发丝

### 问题 3：头发太平

**原因：** 缺少体积感
**解决：** 添加光影，增强立体感

### 问题 4：高光不自然

**原因：** 高光位置或形状不对
**解决：** 根据光源和发型调整
