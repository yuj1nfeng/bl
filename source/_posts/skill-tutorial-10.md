## 转场基础

### 转场的作用

**叙事功能：**
- 场景转换
- 时间跳跃
- 空间变化
- 情绪转换

**视觉功能：**
- 流畅衔接
- 创意表达
- 节奏控制

### 转场原则

**服务内容：**
- 转场为叙事服务
- 不要为了炫技
- 符合影片风格

**保持一致：**
- 转场风格统一
- 不要混用过多类型
- 符合整体调性

## 基础转场技巧

### 硬切（Cut）

**特点：**
- 最常用（90% 以上）
- 干脆利落
- 无特效

**适用：**
- 常规场景切换
- 对话场景
- 动作场景

### 淡入淡出（Fade）

**淡入：** 黑屏→画面
**淡出：** 画面→黑屏

**时长：**
- 快速：0.5-1 秒
- 正常：1-2 秒
- 缓慢：2-4 秒

**适用：**
- 影片开始/结束
- 时间流逝
- 章节分隔

### 叠化（Dissolve）

**特点：**
- 前镜头渐隐
- 后镜头渐显
- 同时发生

**时长：** 1-2 秒

**适用：**
- 时间流逝
- 空间转换
- 回忆幻想

## 创意转场技巧

### 匹配转场

**形状匹配：**
```
镜头 1：圆形物体（太阳）
镜头 2：圆形物体（钟表）
```

**动作匹配：**
```
镜头 1：角色起身
镜头 2：另一角色起身
```

**颜色匹配：**
```
镜头 1：红色物体
镜头 2：红色物体
```

### 遮挡转场

**原理：** 利用前景物体遮挡完成转场

**常见遮挡物：**
- 人物走过镜头
- 车门关闭
- 墙壁
- 柱子

**制作要点：**
- 遮挡物填满画面
- 下一镜头从遮挡开始
- 衔接自然

### 动态转场

**快速摇摄：**
```
镜头 1：快速向右摇
镜头 2：从向左摇开始
```

**物体划过：**
```
快速物体划过屏幕
带动场景转换
```

** zoom 转场：**
```
镜头 1：快速 zoom in
镜头 2：从 zoom out 开始
```

### 声音转场

**声音先入：**
- 下一场景声音提前进入
- 引导观众预期
- 流畅自然

**声音延续：**
- 当前场景声音延续
- 带入下一场景
- 建立联系

**声音相似：**
```
镜头 1：闹钟响
镜头 2：汽笛响
（声音相似转场）
```

## 情绪转场设计

### 紧张情绪转场

**特点：**
- 快速硬切
- 动态转场
- 节奏快

**技巧：**
- 快速摇摄
- 快速 zoom
- 硬切

### 悲伤情绪转场

**特点：**
- 缓慢叠化
- 淡入淡出
- 节奏慢

**技巧：**
- 长叠化
- 缓慢淡出
- 静止转场

### 欢快情绪转场

**特点：**
- 中等速度
- 创意转场
- 有跳跃感

**技巧：**
- 匹配转场
- 动态转场
- 遮挡转场

## 特殊转场效果

### 时间流逝转场

**技巧：**
- 叠化
- 淡入淡出
- 同一场景不同时间

**示例：**
```
镜头 1：白天场景
叠化
镜头 2：夜晚场景
```

### 空间转换转场

**技巧：**
- 匹配转场
- 遮挡转场
- 动态转场

**示例：**
```
镜头 1：角色开门
镜头 2：从门内开始
```

### 梦境转场

**技巧：**
- 柔化叠化
- 扭曲效果
- 颜色变化

**效果：**
- 梦幻感
- 不真实感
- 超现实

## 常见问题

### 问题 1：转场太花

**症状：** 每个转场都用特效

**解决：**
- 多用硬切
- 特效转场用在关键处
- 保持简洁

### 问题 2：转场生硬

**症状：** 转场不流畅，很突兀

**解决：**
- 选择合适转场方式
- 注意节奏匹配
- 添加过渡元素

### 问题 3：转场混乱

**症状：** 转场方式太多，风格不统一

**解决：**
- 确定主要转场方式
- 保持风格一致
- 限制转场类型数量
