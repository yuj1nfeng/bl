## 收入分层

**头部创作者（粉丝 100 万+）：**
```
月收入：50-200 万
收入来源:
- 平台分成：30%
- 广告商单：40%
- IP 授权：20%
- 衍生销售：10%
```

**腰部创作者（粉丝 10-100 万）：**
```
月收入：5-50 万
收入来源:
- 平台分成：40%
- 广告商单：35%
- 衍生销售：15%
- 其他：10%
```

**新人创作者（粉丝 1 万以下）：**
```
月收入：3000-10000 元
收入来源:
- 平台分成：70%
- 少量商单：20%
- 其他：10%
```

## 收入增长路径

**0-1 万粉（3-6 月）：**
- 目标：稳定更新
- 收入：3000-5000 元

**1 万 -10 万粉（6-12 月）：**
- 目标：打造爆款
- 收入：1-5 万元

**10 万 -100 万粉（12-24 月）：**
- 目标：IP 打造
- 收入：5-50 万元

## 增收建议

1. 多平台分发
2. 打造个人 IP
3. 多元化变现
4. 持续学习提升
