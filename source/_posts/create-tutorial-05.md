## 情感线类型

### 按发展阶段

**暗恋阶段：**
- 默默关注
- 不敢表白
- 吃醋心理
- 自我纠结

**暧昧阶段：**
- 若即若离
- 互相试探
- 心跳加速
- 患得患失

**热恋阶段：**
- 甜蜜互动
- 如胶似漆
- 眼里只有对方
- 愿意付出

**磨合阶段：**
- 性格冲突
- 价值观差异
- 争吵和解
- 成长理解

### 按情感类型

**甜宠型：**
- 全程高甜
- 无虐心
- 满足幻想

**虐恋型：**
- 爱而不得
- 误会重重
- 虐心虐身

**成长型：**
- 互相成就
- 共同进步
- 成熟爱情

## 情感推进技巧

### 渐进式推进

**阶段设计：**
```
陌生 → 认识 → 熟悉 → 暧昧 → 表白 → 恋爱
```

**每个阶段：**
- 设置标志性事件
- 情感升级
- 关系变化

### 波折式推进

**波折类型：**
- 误会
- 第三者
- 家庭反对
- 现实阻碍
- 性格冲突

**波折设计原则：**
- 符合人物性格
- 有合理原因
- 最终能解决
- 促进感情发展

### 对比式推进

**情感对比：**
- 前期冷漠 vs 后期热情
- 表面嫌弃 vs 内心在意
- 对他人冷漠 vs 对 ta 温柔

**行为对比：**
```
对别人：公事公办
对 ta：格外关心

对别人：保持距离
对 ta：主动靠近
```

## 情感名场面设计

### 初次心动

**设计要素：**
- 特定场景
- 特殊角度
- 细节特写
- 内心独白

**示例场景：**
- 阳光下的侧脸
- 认真工作的样子
- 不经意的小动作
- 保护 ta 的瞬间

### 第一次接触

**接触类型：**
- 意外触碰
- 主动牵手
- 拥抱
- 壁咚

**描写要点：**
- 触电般的感觉
- 心跳加速
- 时间静止
- 大脑空白

### 表白场景

**表白方式：**
- 直接表白
- 暗示表白
- 行动表白
- 借物表白

**表白要素：**
- 合适的时机
- 真诚的话语
- 紧张的氛围
- 期待的回应

## 情感细节描写

### 肢体语言

**心动表现：**
- 脸红
- 低头
- 玩头发
- 不敢对视

**紧张表现：**
- 手心出汗
- 说话结巴
- 小动作增多
- 呼吸急促

### 心理活动

**暗恋心理：**
```
ta 刚才看我了！
是不是对我有意思？
不对，应该是我想多了...
```

**吃醋心理：**
```
ta 和 ta 聊得真开心
有什么好聊的
哼，不理 ta 了
```

### 环境烘托

**甜蜜场景：**
- 阳光
- 花海
- 夕阳
- 星空

**悲伤场景：**
- 雨天
- 落叶
- 空房间
- 黄昏

## 常见问题

### 问题 1：发展太快

**症状：** 刚认识就爱上，没有过程

**解决：**
- 增加相处时间
- 设计多个阶段
- 让感情自然发展

### 问题 2：误会太假

**症状：** 一句话能说清的事偏不说

**解决：**
- 误会有合理原因
- 符合人物性格
- 尽快解开

### 问题 3：撒糖太硬

**症状：** 为甜而甜，不自然

**解决：**
- 符合情境
- 符合性格
- 有水到渠成的感觉
