## 为什么要做市场调研？

> "知己知彼，百战不殆。"

市场调研能帮助你：

- **了解受众需求** - 知道观众想看什么
- **发现市场机会** - 找到未被满足的需求
- **规避潜在风险** - 避免踩前人踩过的坑
- **明确自身定位** - 找到差异化竞争优势

## 一、行业分析

### 1. 市场规模调研

**数据来源：**
- 行业协会报告（中国动漫产业协会）
- 咨询公司报告（艾瑞、易观）
- 平台公开数据（B 站年报、抖音数据）
- 新闻媒体报导

**关键指标：**
```
市场规模指标:
├── 整体市场规模（亿元）
├── 用户规模（亿人）
├── 增长率（%）
├── 人均消费（元）
└── 付费率（%）
```

**2026 年漫剧市场数据（示例）：**
| 指标 | 数值 | 同比增长 |
|------|------|----------|
| 市场规模 | 210 亿 | +33% |
| 用户规模 | 5.1 亿 | +21% |
| 付费率 | 28% | +5% |
| 人均消费 | 156 元 | +12% |

### 2. 发展趋势分析

**PEST 分析框架：**

```
P（政治）:
□ 文化产业政策支持
□ 内容审核规范
□ 版权保护力度

E（经济）:
□ 文化消费支出增长
□ 广告市场规模
□ 资本投入情况

S（社会）:
□ Z 世代成为主力
□ 二次元文化普及
□ 短视频观看习惯

T（技术）:
□ AI 制作工具普及
□ 虚拟制作技术
□ 5G 传输技术
```

**主要平台政策对比：**

| 平台 | 时长要求 | 内容规范 | 分成政策 | 流量扶持 |
|------|----------|----------|----------|----------|
| B 站 | 1-10 分钟 | 较宽松 | 充电 + 广告 | 新星计划 |

## 二、用户分析

### 1. 用户画像构建

**画像维度：**
```
基础属性:
├── 年龄分布
├── 性别比例
├── 地域分布
├── 职业/学生
└── 收入水平

行为特征:
├── 观看时段
├── 观看设备
├── 观看时长
├── 互动习惯
└── 付费意愿

心理特征:
├── 兴趣爱好
├── 价值观
├── 情感需求
├── 社交需求
└── 自我实现
```

**用户画像模板：**
```markdown
# 用户画像 - 小雅

## 基本信息
- 年龄：22 岁
- 性别：女
- 城市：杭州
- 职业：大学生
- 月可支配：2500 元

## 观看习惯
- 时段：晚上 8-11 点
- 设备：手机为主
- 时长：每天 1-2 小时
- 平台：B 站、抖音

## 内容偏好
- 类型：恋爱、奇幻、治愈
- 风格：精美画风、甜宠剧情
- 时长：3-5 分钟/集

## 付费习惯
- 月均文化消费：300 元
- 付费意愿：中等
- 付费原因：支持喜欢的作品、解锁番外

## 触达渠道
- 社交媒体：微博、小红书
- 信息获取：UP 主推荐、热搜
```

### 2. 用户需求调研

**调研方法：**

**1. 问卷调查**
```
问卷设计要点:
□ 问题数量：10-15 题
□ 填写时间：3-5 分钟
□ 题型搭配：选择 + 开放
□ 样本数量：100+ 份
```

**问卷模板：**
```
漫剧观看习惯调查

1. 您的性别？
   ○ 男 ○ 女 ○ 其他

2. 您的年龄？
   ○ 16 岁以下 ○ 16-22 岁 ○ 23-30 岁 ○ 30 岁以上

3. 您平均每周观看几部漫剧？
   ○ 1-2 部 ○ 3-5 部 ○ 6-10 部 ○ 10 部以上

4. 您最喜欢的漫剧类型？（多选）
   □ 恋爱 □ 奇幻 □ 悬疑 □ 喜剧 □ 励志
   □ 古风 □ 校园 □ 职场 □ 其他____

5. 您选择漫剧的首要因素是？
   ○ 题材 ○ 画风 ○ 剧情 ○ 配音 ○ 推荐

6. 您能接受的单集时长是？
   ○ 1-2 分钟 ○ 3-5 分钟 ○ 5-10 分钟 ○ 10 分钟以上

7. 您有为漫剧付费的经历吗？
   ○ 经常 ○ 偶尔 ○ 很少 ○ 从未

8. 您希望看到但目前较少的是？
   ________________________________
```

**2. 深度访谈**
```
访谈提纲:
1. 您第一次接触漫剧是什么时候？
2. 最喜欢的一部漫剧是什么？为什么？
3. 什么情况下会弃剧？
4. 什么情况下会推荐给别人？
5. 对目前漫剧市场有什么看法？
6. 希望看到什么样的漫剧？
```

**3. 评论分析**
- 收集热门作品评论
- 提取高频关键词
- 分析情感倾向
- 总结用户喜好

### 3. 用户行为分析

**观看行为数据：**
```
关键行为指标:
├── 完播率 - 看完的比例
├── 留存率 - 继续观看下一集
├── 互动率 - 点赞/评论/分享
├── 转化率 - 关注/付费
└── 复看率 - 重复观看
```

**行为洞察：**
- 前 3 秒决定去留
- 15 秒内需要吸引点
- 结尾悬念提升完播
- 互动引导提升参与

## 三、竞品分析

### 1. 竞品选择

**竞品分类：**
```
竞品层级:
├── 直接竞品 - 同类型、同受众
├── 间接竞品 - 不同类型、同受众
└── 潜在竞品 - 可能进入的创作者
```

**选择标准：**
- 同类型 Top 10 作品
- 近期热门新作
- 同体量成功作品
- 失败案例作品


### 5. 差异化定位

**定位矩阵：**
```
              高制作成本
                  │
     精品路线    │    高端小众
                  │
───────┼──────────┼──────────
       │          │
       │ 平民路线  │    性价比路线
       │          │
                  │
             低制作成本

       低 ← 差异化程度 → 高
```

**差异化策略：**

| 策略 | 说明 | 示例 |
|------|------|------|
| 题材差异化 | 选择蓝海题材 | 职场 + 奇幻 |
| 风格差异化 | 独特视觉风格 | 水墨风、像素风 |
| 叙事差异化 | 创新叙事方式 | 倒叙、多视角 |
| 时长差异化 | 非常规时长 | 1 分钟/集、10 分钟/集 |
| 更新差异化 | 特殊更新节奏 | 日更、周更、季播 |

## 总结

市场调研是漫剧项目成功的基石。关键要点：

1. **数据驱动** - 用数据支撑决策
2. **用户视角** - 站在受众角度思考
3. **竞品学习** - 取长补短，差异化竞争
4. **持续跟踪** - 市场在变，调研不停
5. **落地应用** - 调研结果要指导实践
