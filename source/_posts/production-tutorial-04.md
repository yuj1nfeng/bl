## 色彩基础

### 色彩三要素

**色相：** 颜色的种类（红、黄、蓝等）
**饱和度：** 颜色的纯度（鲜艳程度）
**明度：** 颜色的明暗程度

### 色相环

**原色：** 红、黄、蓝（无法调出的颜色）
**间色：** 橙、绿、紫（两种原色调和）
**复色：** 三种以上颜色调和

## 配色方案

### 类似色配色

**定义：** 色相环上相邻的颜色
**效果：** 和谐统一
**应用：** 温馨场景、日常场景

```
示例:
主色：蓝色
辅色：蓝绿、蓝紫
点缀：浅蓝
```

### 互补色配色

**定义：** 色相环上相对的颜色
**效果：** 对比强烈
**应用：** 冲突场景、高潮场景

```
示例:
主色：蓝色
对比：橙色
```

### 三色配色

**定义：** 色相环上等距的三种颜色
**效果：** 丰富平衡
**应用：** 热闹场景、节日场景

```
示例:
红、黄、蓝
```

### 单色配色

**定义：** 同一色相不同明度/饱和度
**效果：** 简洁高级
**应用：** 简约场景、专业场景

## 色彩情感

### 暖色系

**红色：** 热情、危险、喜庆
**橙色：** 活力、温暖、友好
**黄色：** 快乐、光明、警示

### 冷色系

**蓝色：** 冷静、忧郁、科技
**绿色：** 自然、治愈、安全
**紫色：** 神秘、高贵、魔法

### 中性色

**黑色：** 神秘、死亡、高级
**白色：** 纯洁、空白、简约
**灰色：** 中性、沉稳、低调

## 配色技巧

### 60-30-10 法则

```
60% 主色 - 决定整体印象
30% 辅色 - 丰富层次
10% 点缀色 - 画龙点睛
```

### 色彩统一

- 确定主色调
- 所有颜色向主色调靠拢
- 用调整层统一画面

### 色彩对比

- 明度对比
- 饱和度对比
- 色相对比
