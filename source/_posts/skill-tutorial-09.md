## 音效分类

### 动作音效

**定义：** 配合角色动作的声音

**常见类型：**
- 脚步声（不同地面）
- 衣物摩擦声
- 拿取物品声
- 开关门声
- 身体动作声

**设计要点：**
- 与动作精确同步
- 符合材质特性
- 层次丰富

### 环境音效

**定义：** 营造场景氛围的背景声音

**常见类型：**
- 室内环境（空调、钟表）
- 室外环境（街道、鸟鸣）
- 特殊环境（咖啡厅、教室）

**设计要点：**
- 音量适中不抢戏
- 持续铺垫
- 符合场景

### 特殊音效

**定义：** 特殊效果或超自然声音

**常见类型：**
- 魔法效果
- 科幻音效
- 梦幻效果
- 时间效果

**设计要点：**
- 有辨识度
- 符合设定
- 层次丰富

## 音效设计原则

### 真实性原则

**材质匹配：**
- 木地板 vs 瓷砖地
- 布衣 vs 皮衣
- 木门 vs 铁门

**空间匹配：**
- 小房间：混响短
- 大厅堂：混响长
- 室外：几乎无混响

### 层次性原则

**声音层次：**
```
前景：主要音效（清晰）
中景：次要音效（中等）
背景：环境音（模糊）
```

**频率层次：**
- 低频：基础铺垫
- 中频：主要信息
- 高频：细节点缀

### 节奏性原则

**节奏匹配：**
- 快节奏场景：音效密集
- 慢节奏场景：音效稀疏
- 停顿：无声也是音效

## 音效制作技巧

### 分层制作

**示例：脚步声**
```
层 1：鞋底接触声（基础）
层 2：地面材质声（细节）
层 3：衣物摩擦声（伴随）
层 4：空间混响（环境）
```

### 变调处理

**用途：**
- 改变音调创造新音效
- 同一音效多次使用不重复
- 创造特殊效果

**方法：**
- 升高音调：变尖锐
- 降低音调：变低沉

### 时间拉伸

**用途：**
- 放慢：创造厚重感
- 加快：创造紧张感
- 反向：创造特殊效果

## 常见场景音效设计

### 打斗场景

**音效元素：**
- 拳脚声
- 衣物挥舞声
- 撞击声
- 倒地声
- 环境破坏声

**设计要点：**
- 力度感强
- 节奏紧凑
- 层次分明

### 魔法场景

**音效元素：**
- 能量聚集声
- 释放声
- 飞行声
- 爆炸声

**设计要点：**
- 有辨识度
- 符合魔法属性
- 有 buildup 过程

### 恐怖场景

**音效元素：**
- 低频铺垫
- 突然声响
- 诡异声音
- 寂静

**设计要点：**
- 利用寂静
- 突然声响
- 低频压迫

## 音效与画面配合

### 同步技巧

**精确同步：**
- 动作与音效对齐
- 误差在 1 帧内
- 重要音效提前 1 帧

**异步处理：**
- 声音先于画面（预示）
- 声音晚于画面（余韵）

### 音量平衡

**音量层级：**
```
对白：-6dB（最突出）
主音效：-12dB
次音效：-18dB
环境音：-24dB
BGM：-24dB（对白时闪避）
```

## 常见问题

### 问题 1：音效太多

**症状：** 全是声音，没有重点

**解决：**
- 做减法
- 突出主要音效
- 留白

### 问题 2：音效太假

**症状：** 声音与画面不匹配

**解决：**
- 选择合适音效
- 调整材质感
- 添加空间感

### 问题 3：音量不平衡

**症状：** 有的太大声，有的听不见

**解决：**
- 统一音量标准
- 自动化音量控制
- 多设备测试
