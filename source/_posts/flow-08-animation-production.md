## 原画与动画的关系

> "原画决定动作的上限，动画决定动作的下限。"

**原画（Key Animation）：**
- 绘制关键动作帧
- 确定动作的起点、终点、转折点
- 定义动作节奏和力度

**动画（In-between）：**
- 补充原画之间的中间帧
- 保证动作流畅连贯
- 符合原画意图

## 一、动画制作流程

### 完整流程

```
分镜分析 → 律表制作 → 原画设计 → 动画补充 → 上色 → 合成
   ↓          ↓          ↓          ↓        ↓      ↓
 1 天       1-2 天     3-5 天     5-10 天   3-5 天  2-3 天
```

### 传统动画 vs 数字动画

| 维度 | 传统动画 | 数字动画 |
|------|----------|----------|
| 绘制媒介 | 纸/赛璐珞 | 数位板/屏 |
| 修改成本 | 高 | 低 |
| 效率 | 低 | 高 |
| 质感 | 手绘感强 | 干净均匀 |
| 成本 | 高 | 较低 |

## 二、动画基本原理

### 迪士尼 12 条动画法则

**1. 挤压与拉伸（Squash and Stretch）**
```
原理:
物体运动时会产生形变

示例:
- 弹跳的球（落地压扁，弹起拉长）
- 角色跳跃（下蹲压扁，起跳拉长）

要点:
✓ 保持体积不变
✓ 形变程度符合材质
```

**2. 预备动作（Anticipation）**
```
原理:
主要动作前要有预备

示例:
- 出拳前先收拳
- 跳跃前先下蹲
- 投掷先后拉

作用:
✓ 让观众有预期
✓ 增强动作力度
```

**3. 演出布局（Staging）**
```
原理:
清晰呈现想要表达的内容

方法:
- 镜头角度选择
- 光影运用
- 构图安排

要点:
✓ 一个镜头一个重点
✓ 避免信息过载
```

**4. 连续动作与关键动作**
```
连续动作（Straight Ahead）:
- 从第一帧画到最后一帧
- 适合流体、火焰等

关键动作（Pose to Pose）:
- 先画关键帧
- 再补充中间帧
- 适合角色表演
```

**5. 跟随与重叠（Follow Through & Overlapping）**
```
跟随动作:
- 主体停止后附属物继续运动
- 如：头发、衣服、尾巴

重叠动作:
- 身体各部位不同时运动
- 如：转身时头先转，身体后转
```

**6. 缓入缓出（Slow In Slow Out）**
```
原理:
动作开始和结束要慢

帧数分配:
- 开始：帧间距小（慢）
- 中间：帧间距大（快）
- 结束：帧间距小（慢）

效果:
✓ 更自然的运动
✓ 增强重量感
```

**7. 弧线运动（Arcs）**
```
原理:
生物运动多呈弧线

示例:
- 手臂摆动
- 头部转动
- 走路轨迹

要点:
✓ 避免机械直线
✓ 弧线流畅自然
```

**8. 次要动作（Secondary Action）**
```
原理:
在主要动作上添加次要动作

示例:
- 走路时手臂摆动
- 说话时手势
- 跑步时头发飘动

作用:
✓ 丰富表演
✓ 增强真实感
```

**9. 时间控制（Timing）**
```
原理:
帧数决定动作速度和重量

参考:
- 轻快动作：帧数少
- 沉重动作：帧数多
- 快速：2-4 帧
- 正常：6-12 帧
- 慢速：12-24 帧
```

**10. 夸张（Exaggeration）**
```
原理:
适度夸张增强表现力

应用:
- 表情夸张
- 动作夸张
- 形变夸张

要点:
✓ 夸张但不失真
✓ 符合角色性格
```

**11. 扎实绘图（Solid Drawing）**
```
原理:
理解三维空间

要点:
✓ 体积感
✓ 透视准确
✓ 重心稳定
```

**12. 吸引力（Appeal）**
```
原理:
角色和动作要有吸引力

方法:
- 简洁清晰的设计
- 流畅优美的动作
- 富有个性的表演
```

## 三、原画制作

### 原画绘制流程

```
Step 1: 分析分镜
- 理解镜头意图
- 确定动作要点
- 标注关键pose

Step 2: 绘制关键pose
- 起始 pose
- 结束 pose
- 转折 pose

Step 3: 标注律表
- 帧数分配
- 动作说明
- 摄影指示
```

### 关键 pose 设计

**pose 设计要点：**
```
✓ 清晰表达动作意图
✓ 体现角色性格
✓ 有张力（避免平淡）
✓ 符合运动规律
```

**pose 检查：**
```
□ 剪影清晰吗？
□ 重心稳定吗？
□ 线条流畅吗？
□ 表情到位吗？
```

### 律表制作

**律表（Exposure Sheet）结构：**
```
┌─────────────────────────────────────┐

├──┼──┼──┼──┼──┼──┼──┼──┼──┼──┼──┼──┤

├──┼──┼──┼──┼──┼──┼──┼──┼──┼──┼──┼──┤

```


## 四、常见动作绘制

### 走路循环

**关键 pose：**
```
4 个关键 pose:
1. 接触位置（一脚着地）
2. 下降位置（重心最低）
3. 经过位置（两腿交叉）
4. 上升位置（重心最高）

帧数分配:
- 标准走路：12 帧/循环
- 快速走路：8 帧/循环
- 慢速走路：16 帧/循环
```

**动作要点：**
```
✓ 重心上下起伏
✓ 手臂自然摆动
✓ 脚部滚动动作
✓ 身体轻微转动
```

### 跑步循环

**关键 pose：**
```
4 个关键 pose:
1. 接触位置
2. 下降位置
3. 腾空位置
4. 经过位置

帧数分配:
- 标准跑步：8 帧/循环
- 快速跑步：6 帧/循环
- 慢速跑步：12 帧/循环
```

**动作要点：**
```
✓ 腾空阶段
✓ 身体前倾
✓ 手臂摆动幅度大
✓ 重心起伏明显
```

### 口型动画

**基础口型：**
```
8 个基础口型:
A - 闭合（m, b, p）
B - 微张（e, i, ü）
C - 中等（a, ä）
D - 大张（o, u）
E - 咧嘴（ei, ai）
F - 咬唇（f, v）
L - 舌抵上颚（l, n）
W - 圆唇（w, q）
```

**口型同步要点：**
```
✓ 提前 2-3 帧（声音传播延迟）
✓ 重点音节强调
✓ 句尾自然闭合
✓ 不要每帧都变（2-3 帧一切）
```

### 表情变化

**表情过渡：**
```
中性→开心:
- 嘴角上扬（3-5 帧）
- 眼睛弯曲（3-5 帧）
- 眉毛舒展（2-3 帧）

中性→悲伤:
- 嘴角下垂（3-5 帧）
- 眼角下垂（3-5 帧）
- 眉头紧锁（2-3 帧）
```

**表情保持：**
```
- 表情到位后保持 12-24 帧
- 添加微动（呼吸、眨眼）
- 避免完全静止
```

### 眨眼动画

**眨眼结构：**
```
闭眼：4 帧
保持闭合：2 帧
睁眼：4 帧
总计：10 帧
```

**眨眼变化：**
```
快速眨眼：6 帧
慢速眨眼：15 帧
用力眨眼：8 帧（挤压）
```

## 六、上色制作

### 上色流程

```
线稿扫描 → 清稿 → 底色 → 阴影 → 高光 → 检查
```

### 色指定

**色指定表：**
```markdown
# 角色色指定 - 林小满

## 皮肤
- 基础色：#FFE0BD
- 阴影色：#E5C0A0
- 高光色：#FFF0D0

## 头发
- 基础色：#2C1810
- 阴影色：#1A0F0A
- 高光色：#4A2F20

## 眼睛
- 虹膜：#8B4513
- 瞳孔：#1A0F0A
- 高光：#FFFFFF

## 服装
- 上衣：#F5F5DC
- 裙子：#4A5568
- 鞋子：#2D3748
```

### 上色技巧

**平涂要点：**
```
✓ 颜色均匀无笔触
✓ 边缘清晰不出界
✓ 缝隙填满无白点
✓ 分层清晰便于修改
```

**阴影处理：**
```
阴影层次:
- 一层阴影（基础）
- 二层阴影（加深）
- 闭塞阴影（最深）

阴影形状:
- 硬边阴影（赛璐璐）
- 软边阴影（厚涂）
```

## 七、动画检查

### 检查清单

```
动画检查清单:

动作检查:
□ 动作流畅吗？
□ 节奏合理吗？
□ 符合运动规律吗？
□ 重量感对吗？

造型检查:
□ 人物一致吗？
□ 比例准确吗？
□ 透视对吗？
□ 有无变形？

技术检查:
□ 线条干净吗？
□ 颜色准确吗？
□ 分层正确吗？
□ 有无遗漏？

时间检查:
□ 时长准确吗？
□ 节奏对吗？
□ 与配音同步吗？
```

### 常见错误

**1. 抖动**
```
原因:
- 线条不稳定
- 位置偏差大

解决:
- 稳定线条
- 减少帧间差异
```

**2. 漂浮感**
```
原因:
- 缺少重量感
- 缓入缓出不足

解决:
- 增加挤压拉伸
- 调整时间分配
```

**3. 机械感**
```
原因:
- 直线运动
- 匀速运动

解决:
- 改为弧线运动
- 添加缓入缓出
```

## 总结

原画与动画是让角色活起来的关键。关键要点：

1. **理解原理** - 掌握 12 条动画法则
2. **重视原画** - 关键 pose 决定质量
3. **流畅动画** - 中间帧保证流畅度
4. **细节处理** - 口型、表情、眨眼
5. **严格检查** - 确保质量稳定
