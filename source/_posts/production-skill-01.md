## 眼睛结构基础

### 眼睛基本构造

**主要部分：**
- 上眼睑
- 下眼睑
- 虹膜（瞳孔周围有色部分）
- 瞳孔（中心黑色部分）
- 高光（反光点）
- 眼白

### 眼睛比例

**标准比例：**
```
眼睛宽度：1 个单位
眼睛高度：0.5-0.6 个单位
两眼间距：1 个眼睛宽度
```

## 不同风格眼睛

### 写实风格眼睛

**特点：**
- 结构完整
- 细节丰富
- 光影真实

**绘制要点：**
1. 画出眼睑厚度
2. 瞳孔要有深浅变化
3. 高光位置自然
4. 添加睫毛细节

### 卡通风格眼睛

**特点：**
- 简化结构
- 夸张大小
- 色彩明快

**绘制要点：**
1. 放大眼睛比例
2. 简化眼睑结构
3. 高光明显
4. 色彩饱和度高

### 少女漫画眼睛

**特点：**
- 超大眼睛
- 多层高光
- 睫毛浓密

**绘制要点：**
1. 眼睛占脸部 1/3
2. 添加 2-3 个高光点
3. 睫毛分层绘制
4. 虹膜细节丰富

## 眼睛情感表达

### 开心眼睛

**特征：**
- 眼睛弯曲成月牙
- 下眼睑上扬
- 高光明显

**绘制要点：**
```
上眼睑：向下弯曲
下眼睑：向上弯曲
瞳孔：可部分遮挡
高光：大而明显
```

### 悲伤眼睛

**特征：**
- 眼角下垂
- 眼神黯淡
- 可能含泪

**绘制要点：**
```
上眼睑：中间下垂
下眼睑：外侧下垂
瞳孔：放大或缩小
高光：减少或消失
```

### 愤怒眼睛

**特征：**
- 眉毛下压
- 眼神锐利
- 眼睑紧张

**绘制要点：**
```
上眼睑：呈 V 字形
下眼睑：紧绷
瞳孔：缩小
高光：锐利
```

### 惊讶眼睛

**特征：**
- 眼睛睁大
- 瞳孔收缩
- 眼白增多

**绘制要点：**
```
上眼睑：高高扬起
下眼睑：下拉
瞳孔：缩小
高光：明显
```

## 眼睛绘制步骤

### Step 1: 画眼眶

```
1. 确定眼睛位置
2. 画出基本形状
3. 注意透视关系
```

### Step 2: 画眼睑

```
1. 上眼睑加粗
2. 下眼睑细化
3. 注意厚度表现
```

### Step 3: 画虹膜和瞳孔

```
1. 画圆形虹膜
2. 中心画瞳孔
3. 注意被眼睑遮挡部分
```

### Step 4: 画上高光

```
1. 确定光源方向
2. 添加主高光
3. 可添加辅助高光
```

### Step 5: 画睫毛

```
1. 从眼睑边缘生长
2. 呈放射状
3. 注意疏密变化
```

### Step 6: 上色

```
1. 虹膜底色
2. 渐变上色
3. 瞳孔加深
4. 高光留白
```

## 不同视角眼睛

### 正面眼睛

**要点：**
- 对称绘制
- 完整展示
- 注意间距

### 侧面眼睛

**要点：**
- 透视变形
- 只画一只眼
- 注意鼻梁遮挡

### 俯视眼睛

**要点：**
- 上眼睑可见更多
- 眼睛变窄
- 瞳孔上移

### 仰视眼睛

**要点：**
- 下眼睑可见更多
- 眼睛变宽
- 瞳孔下移

## 常见问题

### 问题 1：眼睛无神

**原因：** 高光位置不对或瞳孔太小
**解决：** 调整高光位置，增大瞳孔

### 问题 2：眼睛不对称

**原因：** 没有辅助线
**解决：** 使用中轴线辅助

### 问题 3：眼睛太呆板

**原因：** 太圆太规则
**解决：** 添加变化，打破完美圆形
