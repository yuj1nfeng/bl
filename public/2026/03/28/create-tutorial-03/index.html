好对白的标准

基本要求

符合人物：

  • 符合身份地位
  • 符合性格特点
  • 符合教育背景
  • 符合时代背景

推动剧情：

  • 传递信息
  • 揭示关系
  • 制造冲突
  • 推动发展

自然流畅：

  • 口语化表达
  • 有停顿打断
  • 有潜台词
  • 有情感色彩

常见问题与解决

问题 1：对白太书面

症状： 像念稿，不自然

解决：

  • 大声朗读
  • 录音听效果
  • 加入口语词
  • 允许不完整句

问题 2：信息倾倒

症状： 大段解释性对白

解决：

  • 拆分信息
  • 用动作代替
  • 让观众推测
  • 信任观众智商

问题 3：千人一面

症状： 所有角色说话一样

解决：

  • 为每个角色设计口头禅
  • 区分教育背景
  • 区分性格特点
  • 区分年龄层次
