调色流程概述

标准流程

各阶段目标

一级校正：

  • 校正曝光
  • 调整白平衡
  • 统一基础色彩

二级调色：

  • 局部调整
  • 选择性调色
  • 肤色校正

风格化：

  • 建立视觉风格
  • 强化情绪表达
  • 形成作品特色

一级校正详解

曝光调整

工具：

  • 波形图
  • 直方图
  • 曝光滑块

调整顺序：

  1. 设置黑点
  2. 设置白点
  3. 调整中间调
  4. 检查高光阴影

白平衡校正

校正方法：

  • 使用白平衡吸管
  • 手动调整色温色调
  • 参考示波器

检查要点：

  • 肤色自然
  • 白色纯正
  • 无明显色偏

对比度调整

调整方法：

  • 使用对比度滑块
  • 调整曲线
  • 使用色轮

注意事项：

  • 保留细节
  • 避免过曝欠曝
  • 符合场景情绪

二级调色详解

HSL 限定

使用方法：

  1. 选择颜色范围
  2. 调整限定参数
  3. 查看蒙版
  4. 进行调整

应用场景：

  • 调整特定颜色
  • 肤色保护
  • 背景分离

窗口调整

窗口类型：

  • 圆形窗口
  • 方形窗口
  • 渐变窗口
  • 自定义窗口

跟踪调整：

  • 设置跟踪点
  • 分析运动
  • 调整窗口位置

Qualifier 技巧

肤色限定：

  • 使用肤色预设
  • 微调限定范围
  • 保护肤色自然

天空限定：

  • 限定蓝色范围
  • 调整天空色彩
  • 增强云层细节

风格化技巧

LUT 应用

LUT 类型：

  • 技术 LUT（色彩空间转换）
  • 创意 LUT（风格化）

使用方法：

  1. 一级校正后应用
  2. 调整 LUT 强度
  3. 微调匹配画面

色彩分离

方法：

  • 高光偏暖
  • 阴影偏冷
  • 增强层次

效果：

  • 电影感
  • 色彩丰富
  • 层次分明

特殊效果

柔光效果：

  • 添加柔光层
  • 调整不透明度
  • 营造梦幻感

暗角效果：

  • 添加暗角
  • 聚焦中心
  • 增强氛围

镜头匹配

参考帧设置

选择标准：

  • 曝光正常
  • 色彩准确
  • 代表性强

设置方法：

  • 截取参考帧
  • 并排对比
  • 调整匹配

匹配技巧

目视匹配：

  • 并排显示
  • 肉眼对比
  • 手动调整

数据匹配：

  • 使用示波器
  • 波形对比
  • 精确匹配

常见问题

问题 1：色彩断层

原因： 过度调整

解决：

  • 添加轻微噪点
  • 减少调整幅度
  • 使用 10bit 输出

问题 2：肤色不自然

原因： 整体调整影响

解决：

  • 单独调整肤色
  • 使用 HSL 限定
  • 参考肤色线

问题 3：画面发灰

原因： 对比度不足

解决：

  • 增加对比度
  • 调整 S 曲线
  • 分离高光阴影
