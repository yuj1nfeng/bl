角色绑定基础

什么是角色绑定

定义：

  • 为角色添加骨骼
  • 建立控制体系
  • 便于动画制作

作用：

  • 提高制作效率
  • 保证角色一致
  • 便于批量生产

绑定类型

2D 骨骼绑定：

  • 关节式绑定
  • 网格变形
  • puppet 绑定

3D 绑定：

  • 完整骨骼系统
  • 肌肉系统
  • 面部绑定

骨骼设置

骨骼结构

主要骨骼：

关节设置

关节位置：

  • 符合人体结构
  • 便于动画制作
  • 避免奇怪变形

关节数量：

  • 不宜过多（难控制）
  • 不宜过少（动作僵）
  • 适中最好

控制器设置

控制器类型：

  • IK 控制器（反向运动）
  • FK 控制器（正向运动）
  • 混合控制器

控制器布局：

  • 易于选择
  • 层次清晰
  • 颜色区分

蒙皮设置

权重绘制

权重原理：

  • 骨骼影响范围
  • 影响程度
  • 平滑过渡

绘制技巧：

  • 从大到小
  • 从主到次
  • 逐步细化

权重检查

检查方法：

  • 旋转关节
  • 观察变形
  • 调整权重

常见问题：

  • 权重过大（变形过度）
  • 权重过小（不动）
  • 权重不均（奇怪变形）

表情绑定

绑定方式：

  • 形状键（Blend Shape）
  • 骨骼驱动
  • 混合绑定

表情类型：

  • 基础表情（8 个）
  • 衍生表情
  • 口型绑定

口型绑定

绑定方法：

  • 音素绑定
  • 形状键绑定
  • 自动口型

口型数量：

  • 基础口型（8 个）
  • 过渡口型
  • 特殊口型

动画制作

关键帧动画

制作方法：

  1. 设置关键 pose
  2. 添加中间帧
  3. 调整曲线
  4. 预览检查

曲线调整：

  • 缓入缓出
  • 节奏控制
  • 力度表现

动作循环

循环类型：

  • 走路循环
  • 跑步循环
  • 呼吸循环

循环要点：

  • 首尾一致
  • 动作流畅
  • 节奏稳定

动画分层

分层方法：

  • 基础动作层
  • 次要动作层
  • 细节层

分层好处：

  • 便于修改
  • 便于复用
  • 提高效率

常见问题

问题 1：变形奇怪

原因： 权重绘制不当

解决：

  • 重新绘制权重
  • 添加辅助骨骼
  • 使用矫正形状

问题 2：动作僵硬

原因： 骨骼太少或曲线太直

解决：

  • 添加骨骼
  • 调整动画曲线
  • 添加次要动作

问题 3：效率低下

原因： 绑定复杂或控制不便

解决：

  • 简化绑定
  • 优化控制器
  • 制作动画预设
